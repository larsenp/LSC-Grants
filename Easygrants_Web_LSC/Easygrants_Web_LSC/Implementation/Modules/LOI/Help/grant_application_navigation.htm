Grant Application Navigation

The following table details the sections within the Grant Application section.

 

Link

Description

Home

The Home link allows the user to begin preparing a new NIC and to view your existing NICs and Competition applications.

Main

The Main link provides a summary of the information requested and the application status.

Applicant Information

The Applicant Information link allows the user to enter the following information:

•Applicant Category
•Address
•Telephone/Fax
•Email Address

Project Information

 

 

 

 

 

 

The Project Information link allows the user to view the following project information:

•Service Area
•Estimated Grant Amount

 

Also, the user is required to answer a question related to grant allocation

 

Legal Needs Assessment Data Collection Methods

 

Legal Needs Assessment Data Sources and Tools

 

Intake System Technology

The Intake System Technology form allows user to describe the technology used in Applicant’s intake process.

LEP Plan and Components

The LEP Plan and Components form allows user to state whether the Applicant has a LEP Plan, and to describe the plan.

Training

The Training form allows user to describe what proportion of the staff received different types of training in the last twenty-four months.

Legal Work Management

The Legal Work Management form allows user to describe how frequently the Applicant uses the methods and/or resources listed to ensure that cases and other services are effectively handled.

Casehandling Protocols

The Casehandling Protocols form allows user to indicate the Applicant-wide written protocols adopted to ensure that cases are being handled effectively.

Case Development Activities

The Case Development Activities form allows user to indicate the extent to which the listed case development activities, litigation strategies, and courtroom or litigation activities have been used by the Applicant’s attorneys within the last twenty-four months.

Private Attorney Involvement

The Private Attorney Involvement form allows user to describe the Applicant’s PAI participation.

Methods Used to Recruit Private Attorneys

The Methods Used to Recruit Private Attorneys form allows user to indicate how often the Applicant uses the methods listed to recruit private attorneys.

Methods Used to Retain Private Attorneys Volunteers

The Methods Used to Retain Private Attorney Volunteers form allows user to indicate how often the Applicant uses the methods listed to retain private attorney volunteers.

Board Policies and Practices

The Board Policies and Practices form allows user to describe governing board policies and practices.

Continuity of Operations Planning

Budget (Form D)

The Budget (Form D) link allows the user to capture information on projected expenses, projected revenue, current year expenses and current year revenue.

Actual Staffing Information (Form E-1)

The Actual Staffing Information (Form E-1) link allows the user to capture information on attorneys and other staff for the current year by age, gender, and ethnicity. It also captures information about average years of experience and average salaries.

Current Year Client Services (Form G-1)

The Current Year Client Services (Form G-1) link allows the user to capture information on calendar year 2008 client services.

Current Year Governing/Policy Body Structure (Form F)

The Current Year Governing/Policy Body Structure (Form F) link allows the user to capture information on policy board structure and board members.

Projected Expenditures by Type of Activity (Form G-12)

The Projected Expenditures by Type of Activity (Form G-12) link allows the user to capture information regarding projected calendar year 2009 expenditures for cases, matters, and supporting activities, as well as for cases in each legal area.

PAI Current Year Expenses (Form M)

The PAI Current Year Expenses (Form M) link allows the user to capture information regarding PAI projected expenses.

Technology (Form K)

The Technology (Form K) link allows the user to capture information on staffing, office technology and software applications currently in place.

Technology Budget (Form D-15)

The Technology Budget (Form D-15) form allows user to capture projected LSC and non-LSC expenses for carrying out Applicant’s 2011 Technology Plan.

Organizational Overview

The overview captures a description of the geographical and cultural characteristics of the Applicant’s service area(s), the scope of legal services provided by the Applicant, and the delivery methods and distinctive characteristics of the Applicant’s organization.

List of References

Applicants must provide five professional references for its organization or, in the case of a new organization, provide references for its principals.

Conflicts, Complaints & Performance Evaluations

Applicant must disclose any potential significant conflicts and should describe its capacity to protect against any such conflicts that may arise during the term of the grant or contract.

Uploads

The Uploads link allows the user to upload documents required for the Grant application.

Review and Submit

The Review and Submit link allows the user to view a PDF of their submission. Also, users can see whether they have completed the application. If they have, they can submit the NIC from this page.

Help

The Help link provides access to detailed instructions and additional information for each section of the system.

Log Out

The Log Out link is used to exit the system.
