Form D-1: Actual Expenses

 Top 

The Form D-1: Actual Expenses page captures information on actual calendar-year 2011 expenditures of LSC and non-LSC funds. Please report expenditures of LSC and non-LSC funds separately for each service area, including expenditures of carryover funds and derivative income in the LSC and non-LSC columns where applicable. The information which you submit in this form does not need to be audited. If you do not have audited expense information at the time when you complete this form, please use unaudited information.

 

Please note that this form will not save properly with non-numeric data such as letters or spaces. Commas and decimal points may be used but are not required. Leaving a numeric cell blank will result in a zero being saved for that value. If you get an error message when entering this information, please note the message on the screen and correct it accordingly.
