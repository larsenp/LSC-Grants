Form G-6: Self-Inspection Unreported Non-LSC Case

 Top 

Form G-6: Self-Inspection captures information on the percentage of sampled CSR cases that have reporting errors, what types of errors occurred, and how many non-LSC funded cases were closed that were not reported to LSC. Form G-6 contains three sections: Certification, Summary and Unreported Non-LSC Cases. These sections are attached to Program Letter 11-2, as are the Case Review Form and the Sample Selection Procedure.

 

The Self-Inspection procedure is set out in Program Letter 10-4. Once the Self-Inspection is completed, the results should be entered in this form, just as they were entered in the hard copy Self-Inspection in prior years.

 

For the Self-Inspection Unreported Non-LSC Cases section below, please be certain to enter a number, even if it is 0, in the applicable field in #1. If the number of cases in #1 is 0, no further entries are necessary. If the number of cases in #1 is greater than zero, please select one and only one of (a), (b), or (c) in question 2. If you selected (c), then please fill in integers greater than zero in the two fields at the end of question 2.

 

Please note the following:

 

•To save data, click Save and Calculate.
•To continue to the next page, click Save and Continue.
