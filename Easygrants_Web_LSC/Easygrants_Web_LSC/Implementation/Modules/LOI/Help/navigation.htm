The following table details the sections within the Notice of Intent to Compete (NIC) section.

 

Link

Description

Home

The Home page allows the user to begin preparing a new Notice of Intent to Compete (NIC) and to view your existing NICs and Competition applications.

Applicant Information

The Applicant Information page allows the user to enter the following information:

•Applicant Category
•Address
•Telephone/Fax
•Email Address

Project Information

The Project Information page allows the user to view the following project information:

•Service Area
•Estimated Grant Amount

 

Also, the user is required to answer a question related to grant allocation

Key Staff Members

The Key Staff Members page allows the user to enter information about their key staff members.

Board Members

The Board Members page allows the user to enter information about their board members.

Review and Submit

The Review and Submit page allows the user to view a PDF of their submission. Also, users can see whether they have completed the application. If they have, they can submit the NIC from this page.

Help

The Help link provides access to detailed instructions and additional information for each section of the system.

Log Out

The Log Out link is used to exit the system.
