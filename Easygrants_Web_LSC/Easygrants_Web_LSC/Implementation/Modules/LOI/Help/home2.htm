•The Grant Activity Reports section below contains a list of all the grantee activity report (GAR) forms assigned to you for submission.
•Grantees with multiple service areas will see links at the bottom right of the list labeled Next 22 >, Last 22>>, etc. Use these links to navigate through the list of forms assigned to you. You can also use the Show All link to display all of your forms at once.
•Click on the links in the Task column to access your forms.
•Each form consists of one or more data entry pages and a Review and Submit page.
•In order to submit your form, you will have to have provided certain required data elements. The Review and Submit page for each form indicates what the requirements are and whether they have been met. If submission requirements have not been met, the Review and Submit page will display an explanation of each unmet requirement.
•To see all of the GAR Instructions, click Help located within the left navigation of each webpage and then double-click Grant Activity Report Instructions.
