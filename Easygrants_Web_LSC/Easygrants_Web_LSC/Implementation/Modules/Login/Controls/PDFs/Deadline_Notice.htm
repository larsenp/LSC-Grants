FY2002 Competition: Application Deadline

The deadline for filing grant applications for service areas in Alabama, Arizona, California District of Columbia, Florida, Georgia, Illinois, Kansas, Massachusetts, Mississippi, Montana, Nevada, New Jersey, New York, Virginia, and West Virginia was June 18, 2001. LSC is no longer accepting grant applications for service areas in those states.
