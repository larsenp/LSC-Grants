Form C Grant Assurances Form 

The LSC grant assurances for 2006 grants will be available soon.  Applicants for 2006 grant renewal and competitive grant funds should file all parts of the grant application except Form C and Form I (Certification Form).  

When the grant assurances are published, LSC will send an e-mail to all applicants informing them of its availability and instructions on how to proceed.  Applicants will be given the time necessary to review Form C and file Form I after Form C is published.

Please contact the competition service desk at competition@lsc.gov, if you have any questions about this matter, or the competition process.
