LSC PROGRAM DIRECTORY

To view a list of programs in a particular state:
(1) select a state from the pulldown box, (2) press "Proceed".
To locate a program which serves a particular county in a particular state:
(1) select a state, (2) select a county, (3) press "Proceed".
To view the entire Program Directory:
(1) select All states from the pulldown box, (2) press "Proceed".
