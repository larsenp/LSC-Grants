The LSC logo is available to download in the following formats:

.jpg (standard) should be used at current or reduced sizes

.eps (vector) can be enlarged and reduced in size

.png (for web-based work)

The .eps and .jpg files are 300 DPI high-resolution images.

The .png files are 72 dpi, low-resolution images for web page layouts only.

We also have versions of each with the words "America's Partner For Equal Justice" and versions with just the logo and letters LSC.

To download click on the format of your choice.
