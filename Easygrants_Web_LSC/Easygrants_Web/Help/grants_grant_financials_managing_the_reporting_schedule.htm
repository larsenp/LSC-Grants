Managing the Reporting Schedule

 

   

From the Reporting Schedule section of the Grant Financials page, staff can add or modify the required reporting submissions, e.g., progress reports, interim reports, expenditure reports, and final reports. The reporting schedule can be created either manually directly from the grant file or based on the designated schedule for the funding cycle, e.g., all grants for the 2008 funding cycle will have two required progress reports and a final report.

 

To add a grantee report record:

 

1.Click on the Add button in the Reporting Schedule section of the Grant Financials page. The Report Editor page is displayed.
2.Click on the Select button if a new assignee is required. The assigned person will default to the primary contact/PI for the grant.
a)In Step 1 of the Person Selection pop-up window, enter part of the person’s last name and click on the Find button.
b)In Step 2, select the person’s name from the drop-down list and click on the Select button. The page will refresh and display the selected person’s name.
3.Select the Workflow Task (required). The list consists of all Grantee Reporting tasks set up in the workflow for the funding cycle.
4.Enter the Start Date.
5.Enter the Open Date.
6.Enter the Due Date.
7.Enter the Close Date.
8.Enter the End Date.

 

If the dates for the task assignment are the same as the task dates displayed, the dates do not need to be entered. The task will default to use the task dates.

 

9.Click on the Save button to save the information and remain on the Report Editor page.
10.Click on the Save and Close button to save the information and return to the Grant Financials page.
11.To return to the Grant Financials page without saving the changes, click on the Close button.

 

Saving the page without entering the required information or with incorrect formatting will display a page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a report, click on the Edit button next to the desired record in the Reporting Schedule section of the Grant Financials page. The Report Editor page is displayed with the selected record information. Use the steps above to complete the modification of the reporting requirement.

 

To delete a report:

 

1.Click on the Delete button to the right of the desired record in the Reporting Schedule section of the Grant Financials page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

Only reports that are incomplete can be deleted from the schedule. If the report has been submitted by the grantee, the Delete button will not be displayed.
