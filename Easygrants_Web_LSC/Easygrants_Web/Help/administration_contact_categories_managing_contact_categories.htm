Managing Contact Categories

 

   

To add a contact category:

 

1.Click on the Add button in the Contact Category section of the Contact Category page. The Contact Category Editor page is displayed.
2.Enter the Category name (required).
3.Enter the Description.
4.Search for person contacts to add to the category by entering the Last Name and clicking the Find button.

 

•The list of available persons is comprised of all contacts in the system matching the last name entered in the search. If you cannot locate the desired person contact, the person is not a contact in the system. First, check to see if the person is a contact in the system using the Contacts Easylist Search. Refer to the section Searching for a Contact for instructions on these search processes. If the person is not a contact in the system, follow the instructions in the section Adding Person Records.
•To add a person, click on the desired name from the Available list box and click on the Add button. The name(s) will be added to the Selected list box.
•To add several names listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting names in a row or the <CTRL> key if selecting names throughout the list. Click on the Add button until all desired names have all been added to the Selected list box.
•To remove previously selected names from the Selected list box, select the name and click on the Remove button. The name(s) will be removed from the Selected list box.

 

5.Click on the Save button after all selections have been made to save the contact category and remain on Contact Categories Editor page.
6.Click on the Save and Close button to save the information and return to the Contact Category page.
7.To return to the Contact Category page without saving changes, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a contact category, click on the Edit button next to the desired record in the Contact Category section of the Contact Category page. The Contact Category Editor page is displayed with the selected record information. Use the steps above to complete the modification of the contact category.

 

To delete a contact category:

 

1.Click on the Delete button to the right of the desired record in the Contact Category section of the Contact Category page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
