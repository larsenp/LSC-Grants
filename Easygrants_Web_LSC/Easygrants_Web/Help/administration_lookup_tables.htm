Lookup Tables

 

   

The Lookup Tables section allows the maintenance of the checklist or drop-down values used throughout Easygrants.

 

For example, the prefixes available on the Add or Edit Person Contact records are maintained using the c_PREFIX table. Values to this table can be updated, added, or marked as inactive. Inactive values will no longer display in the drop-down. These values are not deleted so that historical information can be retained.

 

       

To access the Lookup Table Management functionality, click on the Administration link in the left navigation. The Program Management page is displayed. Click on the Lookup Tables link. The  Lookup Table Management page is displayed.

 

All lookup tables available for management by administration users are displayed. The lookup tables can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.
