Creating a New Query

 

   

The Create New Query section of the Query/Report List page allows the user to create new dynamic reports for the data collected in Easygrants. Queries are created using a specific query field group. Query field groups break down the Easygrants data into smaller groupings to allow easier querying capability.

 

To create a new query:

 

1.Select the Field group.

 

To view the fields available in a particular field group, click the here link below the query field. Select the field group and click on the Go button. The fields for the selected field group will be displayed with the description and the merge code. The merge codes can be used in e-mail templates or Word Merge documents.

 

2.Click on the Go button. The Query Tool Editor page is displayed. Use the instructions in the related sections on selecting, sorting, filtering, viewing, and saving the query results.
