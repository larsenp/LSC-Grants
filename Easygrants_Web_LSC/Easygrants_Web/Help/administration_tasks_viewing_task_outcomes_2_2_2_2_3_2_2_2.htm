Entering the Submission Validation

 

   

Each form for the tasks included in the workflow can have associated requirements for submission (validation).

 

Example: In order to submit the application task, the user must complete the following fields on the project information page, Project Title, Brief Project Description, and Request Amount.

 

Note: The page level validation (requirements to save the page such as the entry of a field or the format of an entry) are set at the form level.

 

To enter the submission validation for the task:

 

1.Click on the Validation button next to the desired task in the Tasks section. The Workflow Builder page is displayed with the task information as read only and a listing of all the associated forms.
2.Click on the Validation button next to the desired form in the Forms section. The Workflow Builder page is displayed.
3.Click on the View Form Validation to view the submission validation for the selected form. The validation is displayed in a new window.
4.Enter the Submission Validation. Submission Validation is entered using XML.  For example, the Grant Title is required for submission of the application, the project information form would have XML validation that reads:

 

       <ValidationItem>

                         <Requirement PropertyKey='GranteeProject' >

                                   <Filters>

                                     <Argument PropertyKey='GrantTitle' Value=''/>

                                   </Filters>

                       </Requirement>

                 <Results>

                     <Result> ResultCode='3'        Message='Title was not entered.'        MinCount='1'/>

                 </Result>        

         </Results>

 

Any validation entered at the workflow level will be in addition to the submission level validation entered at the task level. Refer to the Task Builder section for more details on adding submission level validation for a selected form.

 

5.Click on the Save button to save the information and remain on the Workflow Builder page.
6.Click on the Save and Close button to save the information and return to the Workflow Builder page.
7.To return to the Workflow Builder page without saving the changes, click on the Close button.
8.Click on the Close button to return to the Workflow Builder Validation page.

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.
