Submitting Internal Review

 

   

The Internal Review process is typically completed by staff members to determine the outcome of a submission, e.g., Submit Letter of Inquiry or Submit Progress Report. Internal Review tasks are accessible from the Home page in either the Workgroup Tasks or Personal Tasks sections. If the task has been assigned to a workgroup, the task should be transferred to your personal tasks. See the Transferring a Workgroup Tasks section for more details.

 

Specific instructions related to the internal review task will be displayed in the Instruction section of the internal review page.

 

To complete the internal review:

 

1.Access the internal review submission by clicking on the Task name link in the Personal Tasks section of the Home page.
2.Review the Grant Information.

 

Clicking the hyperlinks in the Grant Information section will redirect the user to the appropriate details page, e.g., clicking the Easygrants ID will redirect the user to the Grant Details page of the related Grant File.

 

3.Review the Task Information. The information displayed in this section is related to the task that is being reviewed.
4.Click on the View button to view the related full submission PDF.
5.Review the Summary Documents. Click on View to access the full submission PDFs previously submitted for the related grant. Note: If this is the first internal review of a submission, e.g., Review Letter of Inquiry, there will be no summary documents displayed.
6.Review the Submitted Documents. The documents displayed in this section are the documents uploaded for the submission. Clicking on the View button will display the file in the originally submitted format, e.g., Microsoft Word.
7.Request a Resubmission on portions of the task being reviewed. Click on Add to access the Resubmission page. See the Resubmission section for more details.
8.Select the Outcome for the review task (required).
9.Select the Outcome for the task being reviewed (e.g., Submit Proposal).
10.Enter Comments.
11.Click on the Save button to save the changes.
12.Click on the Save and Close button to save the changes and return to the Home page.
13.Click on the Close button to return the Home page without saving any changes.
14.Click on the Submit button to submit the internal review. This action will also save any changes on the page. Once the task has been submitted, it will be removed from the Personal Tasks list.

 

Saving the page without entering required information or with incorrect formatting will display a page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

Based on the designated workflow for the funding cycle and the selected outcome, submitting the internal review task can create additional task assignments, send e-mails to related staff or public users, or update the outcomes for other task assignments.
