Creating the Eligibility Quiz

 

   

The next step is to associate the questions that are completed by the potential grantee to determine their eligibility to apply for the selected funding opportunity.

 

To select the questions to be associated with the task:

 

1.Click on the Quiz button next to the Eligibility Task. The Workflow Builder - Eligibility Quiz page is displayed.

 

For the eligibility questions, a template can be used or questions can be added manually. If selecting a template first, additional questions can be added to the quiz.

 

To associate the questions from a template:

 

1.Select the template as needed. The page will refresh and display the questions and required answers in the Eligibility Questions section.
2.To return the Eligibility Workflow Builder page, click on the Close button.

 

To create an Eligibility Quiz template, click on the Eligibility link in the left navigation.

 

Selecting another template from the drop-down will remove any previously added questions.

 

To add questions manually:

 

1.Click on the Add button in the Eligibility Questions section. The Workflow Builder - Eligibility Quiz Editor page is displayed.
2.Select the Question (required).
3.Select the Valid Answer (required).
4.Click on the Save button to save the information and remain on the page.
5.Click on the Save and Continue button to save the information and return to the Workflow Builder - Eligibility Quiz page.
6.To return the Workflow Builder - Eligibility Quiz page without saving the changes, click on the Close button.

 

The eligibility questions and answers can be added using the Lookup Table Management. Refer to the Lookup Tables section for more details.

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a question, click on the Edit button next to the desired record in the Eligibility Questions section. The Workflow Builder - Eligibility Quiz Editor page is displayed with the selected record information. Use the steps above to complete the modification of the task.

 

To delete a question:

 

1.Click on the Delete button next to the desired record in the Eligibility Questions section of the Eligibility Workflow -  Eligibility Quiz page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

Eligibility questions and answers can be modified at any time. After the modification the eligibility task will display the updated quiz. Any potential grantees that have already been deemed eligible will remain eligible.
