Creating Eligibility Tasks

   

To add a new eligibility task:

 

1.Click on the Add button in the Task Builder (Eligibility) section of the Task Builder page. The Task Builder Eligibility Editor page will be displayed. The Type of the task will be displayed as read only.
2.Enter the Task Name (required).
3.Enter the Description (required).
4.Enter the Guideline File URL. If the URL is external to the Easygrants project, check the External URL box.
5.Enter the Help File URL. If the URL is external to the Easygrants project, check the External URL box.
6.Click on the Save button to save the information and remain on the Task Builder Eligibility Editor page. The outcomes section will display Eligible and Not Eligible.
7.Click on the Save and Close button to save the information and return to the Task Builder page.
8.To return to the Task Builder page without saving the changes, click on the Close button.

 

To modify the outcomes for an eligibility task:

 

1.Click on the Edit button next to the desired record in the Outcomes section of the Task Builder Eligibility Editor page. The Task Builder Outcomes Editor page will be displayed. The Task Name will be displayed as read only.
2.Modify the Outcome (required).
3.Click on the Save button to save the information and remain on the Task Builder Outcomes Editor page.
4.Click on the Save and Close button to save the information and return to the Task Builder Eligibility Editor page.
5.To return to the Task Builder Eligibility Editor page without saving the changes, click on the Close button.

 

To modify a selected task outcome, click on the Edit button for the desired record and follow the steps above. To delete the outcome from the task, click on the Delete button for the desired record.

 

To edit an existing eligibility task, click on the Edit button next to the desired record in the Task Builder (Eligibility) section of the Task Builder page. The Task  Builder Eligibility Editor page is displayed with the selected record information. Use the steps above to complete the modification of the task.

 

To delete an eligibility task:

 

1.Click on the Delete button next to the desired record in the Task Builder (Eligibility) section of the Task Builder page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

The Delete button is only available for tasks that have not been associated to a workflow.

 

To copy an eligibility task:

 

1.Click on the Copy button next to the desired record in the Task Builder (Eligibility) section of the Task Builder page.
2.Click on the OK button in the confirmation pop-up to continue with the copy. The Copy Task page will be displayed.
3.Modify the New Task Name (required). The task name must be unique in order to finish copying the task.
4.Modify the Description (required).
5.Click on the Copy button to continue copying the task.
6.To return to the Task Builder page without saving the changes, click on the Close button.
