Managing Scores and Comments by Reviewer

   

Review scores and comments can be viewed and updated by reviewer within the View and Modify Conflicts/Preferences, Scores, and Comments section of the Review Stage Scores and Comments page of the review stage.

 

To view the assignments and submission status for a reviewer:

 

1.Select the Reviewer name next to the By Reviewer field.
2.Click on the Go button. The Scores and Comments by Reviewer page is displayed.

 

The Scores and Comments by Reviewer page displays the selected reviewer information and the assignments (including the total number of assignments) in the View and Modify Conflicts/Preferences, Scores, and Comments section with the following columns:

 

•Select (checkbox)
•Participant
•Participant Organization
•Assignment Type
•Status
•Conflicts/Preferences
•Score
•Options

 

The results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

In the View and Modify Conflicts/Preferences, Scores, and Comments section, conflicts and preferences and submission results can be modified. In addition, the user can select using the Select checkbox submissions that need to be submitted or unsubmitted by staff for the reviewer.

 

Unsubmitting a review assignment for the reviewer will allow the reviewer to access the task on their Home page. To remove an assignment for a reviewer, refer to the Managing Review Assignments by Reviewer section.

 

To return to the Review Stage Scores and Comments page for the review stage, click on the Close button.

 

To submit review assignments by reviewer:

 

1.Select the review assignments that need to be submitted by clicking on the checkbox in the Select column in the View and Modify Conflicts/Preferences, Scores, and Comments section.

 

To select or unselect all review assignments, click the Check All link or Clear All link in the Submit or Unsubmit for Reviewer section.

 

2.Click on the Go button next to Submit Checked Reviews in the Submit or Unsubmit for Reviewer section. The page will refresh displaying the updated status in the Status column of the View and Modify Conflicts/Preferences, Scores, and Comments section.

 

Reviewers will not be able to access submitted review assignments (assignments with a status of complete).

 

To unsubmit review assignments by reviewer:

 

1.Select the review assignments that need to be unsubmitted by clicking on the checkbox in the Select column in the View and Modify Conflicts/Preferences, Scores, and Comments section.

 

To select or unselect all review assignments, click the Check All link or Clear All link in the Submit or Unsubmit for Reviewer section.

 

2.Click on the Go button next to Unsubmit Checked Reviews in the Submit or Unsubmit for Reviewer section. The page will refresh displaying the updated status in the Status column of the View and Modify Conflicts/Preferences, Scores, and Comments section.

 

Modifying existing review assignment conflicts/preferences and scores and comments can be done in bulk or one at a time.

 

To edit all external (noted by the reviewer) conflicts and preferences for a reviewer:

 

1.Click on the Edit All External Conflicts and Preferences button in the View and Modify Conflicts/Preferences, Scores, and Comments section.. The  External Conflicts and Preferences Editor page is displayed.
2.Select the Conflict/Preference Type from the list, Able, Not Able, Conflict or Preferred.
3.Select the Conflict Type from the drop-down.
4.Click on the Save button to save the information and remain on the External Conflicts and Preferences Editor page.
5.Click on the Save and Close button to save the information and return to the Scores and Comments by Reviewer page.
6.To return to the Scores and Comments by Reviewer page without saving changes to the information, click on the Close button.

 

To edit all scores and comments for a reviewer:

 

1.Click on the Edit All Scores button in the View and Modify Conflicts/Preferences, Scores, and Comments section. The All Scores Editor page is displayed.
2.Enter the Scores for each participant.
3.Click on the Edit button for a particular participant to add or modify the comments. The Score and Comments Editor page will be displayed.
4.Click on the Save button to save the information and remain on the All Scores Editor page.
5.Click on the Save and Close button to save the information and return to the Scores and Comments by Reviewer page.
6.To return to the Scores and Comments by Reviewer page without saving changes to the information, click on the Close button.

 

To edit a single score or comment for a reviewer:

 

1.Click on the Edit button next to the desired participant record being reviewed. The Scores and Comments Editor page is displayed. The review assignment information will be displayed as read only.
2.Modify the Conflict and Preference selecting the Conflict/Preference and the Conflict Type (Financial, Organizational, Personal).
3.Modify the Score.
4.Modify the Reviewer Comments.
5.To view the Reviewer Comments Upload, click the View PDF button.

 

If no PDF is available, the system will display a message that the requested PDF is not available. This will either be displayed if not PDF was uploaded by the reviewer or the task does not require an upload.

 

6.Modify the Staff Edited Comments.

 

Staff Edited Comments are used to provide feedback to the participant.

 

7.Click on the Save button to save the information and remain on the Scores and Comments Editor page.
8.Click on the Save and Close button to save the information and return to the Scores and Comments by Reviewer page.
9.To return to the Scores and Comments by Reviewer page without saving changes to the information, click on the Close button.
