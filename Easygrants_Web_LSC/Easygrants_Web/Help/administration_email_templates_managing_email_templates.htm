Managing E-mail Templates

 

   

To add an e-mail template:

 

1.Click on the Add button in the Add or Edit E-mail Templates section of the E-mail Templates page. The E-mail Template Editor page is displayed.
2.Enter the Template Name (required).
3.Enter the Template Description.
4.Select the Query Field Group.

 

The Query Field Group selected will determine where the template is available for use in the Easygrants system.

 

5.Enter the From Address (required).
6.Enter the To Address (required).
7.Enter the CC Addresses.
8.Enter the BCC Addresses.

 

An e-mail will be sent to the CC and BCC e-mail address for each recipient on the list. For example, if you are sending five e-mails, five e-mails will be sent to each address in the CC and BCC fields.

 

9.Enter the Subject (required).
10.Select if the template uses HTML formatting. Selecting Yes will allow HTML formatting using HTML tags, e.g., <b>Bold</b>.

 

If HTML Format is selected for the e-mail template, the merge codes need to be prefaced by “eg” and end with a forward slash (Example: [egWfTaskAssignment.GranteeProject.AmountAwarded{$#,0.00}/]).

 

If HTML Format is not selected, remove “eg” and the forward slash from the merge codes (Example: [WfTaskAssignment.GranteeProject.AmountAwarded{$#,0.00}]).

 

11.Select if the template is logged. Selecting Yes will log the e-mail and display the e-mail in the To person contact's Correspondence tab.
12.Enter the Text (required).

 

To view the merge codes that can be used in the e-mail template, click on the Query Fields and Merge Codes link in the instructional text on the E-mail Template Editor page. The Query Tool Field Groups and Merge Codes page is displayed in a new window. Select the Field Group from the drop-down and click on the Go button. The page will refresh and display the field names available with the related description and merge code.

 

To specify date and time formatting for a merge code, refer to the following examples:

 

• [egWfTaskAssignment.GranteeProject.GrantApprovalDate{M/dd/yyyy h:mm a}/] displays a Grant Approval Date such as 10/15/2009 10:25 am
•{M/dd/yyyy h:mm a} = 10/15/2009 10:25 am
•{M/dd/yyyy} = 10/15/2009
•{MMMM dd, yyyy} = October 15, 2009

 

To specify currency formatting for a merge code, refer to the following examples:

 

•[egWfTaskAssignment.GranteeProject.AmountAwarded{$#,0.00}/] displays an Amount Awarded such as $1,500.99
•{$#,0.00} = $1,500.00
•{#,0.00} = 1,500.00
•{#,0} = 1,500
13.Click on the Save button to save the information and remain on the E-mail Template Editor page.
14.Click on the Save and Close button to save the information and return to the E-mail Templates page.
15.To return to the E-mail Templates page without saving the changes, click on the Close button.

 

Saving the page without entering required information or incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit an e-mail template, click on the Edit button next to the desired record in the Add or Edit E-mail Templates section or the Edit System E-mail Templates  section of the E-mail Templates page. The  E-mail Template Editor page is displayed with the selected record information. Use the steps above to complete the modification of the template.

 

To delete an e-mail template:

1.Click on the Delete button to the right of the desired record in the Add or Edit E-mail Templates section of the E-mail Templates page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
