Historical Import Requests

 

   

The Historical Import Requests Listing page allows the user to view previously imported requests.

 

To view historical request data:

       

1.Click on the View Historical Requests button in the Historical Import Requests section of the Import Management page. Historical Import Requests page is displayed.
2.Click on the View button to view the records that were imported in a new window.
3.To return to the Import Management page, click on the Close button.
