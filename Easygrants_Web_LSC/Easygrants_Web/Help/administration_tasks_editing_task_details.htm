Review Workflow Builder

   

The Review Workflow Builder allows the user to select review type tasks (e.g., Submit Conflicts and Preferences, Submit Preliminary Review Scores) and internal review tasks (e.g., Review Letter of Intent) to be completed as part of the workflow.

 

Internal Review type tasks must be associated with the task that is being reviewed. For example, if there is an internal review of the Submit Letter of Intent, the internal review task must be added to the Application Workflow Builder rather than the Review Workflow Builder. In most cases, the only type of tasks being added in the Review Workflow Builder will be External review type tasks.

 

Review type tasks are created in the Task Builder. Refer to the Task Builder section for more details.

 

To access the Review Workflow Builder page:

 

1.Click on the View button next to the desired Funding Cycle in the Funding Cycle Management section of the page.
2.The page refreshes and displays the workflow builder tabs for each grant lifecycle phase starting with the Eligibility Workflow Builder.
3.Click on the Review tab.

 

 

The Review Workflow Builder is comprised of the following steps:

 

1.Selecting the tasks to be completed in the review phase either by the public user or by internal staff.
2.Determining the review workflow based on the outcome of the related tasks.

 

Assigning the external review tasks (Submit Conflicts and Preferences, Submit Preliminary Review Scores) is done through the Review Processing module of the Easygrants module.
