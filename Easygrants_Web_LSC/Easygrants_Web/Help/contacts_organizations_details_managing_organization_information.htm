Managing Organization Information (Summary)

 

   

The organization name and primary information can be managed from the Summary section on the Organization Details page. Updating any telephone or address information will update that information in the appropriate information sections as well.

 

To edit the organization:

       

1.Click on the Edit button in the Summary section of the Organization Details page. The Organization Editor page is displayed.
2.Modify the Legal Organization Name (required).
3.Modify the Common Organization Name (required).
4.Select the Type.
5.Modify the Employer Identification Number (EIN).
6.Modify the Address Line 1 (required), Address Line 2 and Address Line 3.
7.Modify the City (required).
8.Modify the State/Province (required).
9.Modify the Region. The region field is used for internal addresses. Select N/A for the state/province and enter the region.
10.Modify the Zip/Postal Code (required).
11.Modify the Country (required).
12.Modify the Phone Number.
13.Modify the Web site Address.
14.Select if the organization is available for public selection. This will make the organization available in the Organization Selector used by public users.
15.Select the Eligibility Status.
16.Click on the Save button to save the information and remain on the Organization Editor page.
17.Click on the Save and Close button to save the information and return to the Organization Details page.
18.To return to the Organization Details page without saving the changes, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

Organizations can be deleted from the Summary section of the Organization Details page. The Delete button is ONLY displayed if the organization is not associated with any grant records or person records related to a grant.

 

To delete the organization:

 

1.Click on the Delete button in the Summary section of the Organization Details page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

   

Deleting the organization is a hard delete. A history of the deletion action will exist only in the Database Log table.
