Managing FAQs

 

   

To add a new FAQ:

 

 

1.Click on the Add button in the FAQ Manager section of the FAQ Manager page. The FAQ Editor page displays.
2.Enter the Question (required).
3.Enter the Answer (required).
4.Check whether the FAQ should be available to public and/or staff users.
5.Click Save to save your changes and remain on the FAQ Editor page.
6.Click Save and Close to save your changes and return to the FAQ Manager page.
7.Click Close to return to the FAQ Manager page without saving your changes.

 

To edit an FAQ, click on the Edit button next to the desired record on the FAQ Manager page. The FAQ Editor page is displayed with the selected record information. Use the steps above to complete the modification of the FAQ.

 

To delete an FAQ:

 

1.Click on the Delete button to the right of the desired record in the FAQ Manager section of the FAQ Manager page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
