Setting and Finalizing Review Stage Outcomes

 

   

The final step for a review stage is to set proposed and final stage or funding outcomes for the participants.

 

To access the Review Stage Outcomes page:

 

1.Click on the Reviews link in the left navigation.
2.Click on the Review Processing link. The Review Cycles page is displayed.
3.Click on the Manage button for a specific review cycle. The Participant/Committee Pool page is displayed.
4.Click on the Review Stages tab in the review cycle. The Review Stages page is displayed.
5.Click on the Manage button for a specific review stage. The Review Stage Assignments page is displayed.
6.Click on the Outcomes tab in the review stage. The Review Stage Outcomes page is displayed.

 

Setting the outcomes for the review stage is a multi-step process. The first step is to select the participants. Participants can be selected one by one, by average score, by rank order, or all at once.

 

To select participants one by one:

 

1.Select the participants that need to be included by clicking on the checkbox next to the record in the Step 2: Set and Finalize Stage Outcomes; Update Recommended Amounts section.

 

To select participants by average score:

 

2.Enter the Average Score range in the Step 1: Select Participants section.
3.Click on the Select button. The page will refresh with all participants matching the entered criteria as checked in the Step 2: Set and Finalize Stage Outcomes; Update Recommended Amounts section.

 

To select participants by rank order:

 

4.Enter the Rank Order range in the Step 1: Select Participants section.
5.Click on the Select button. The page will refresh with all participants matching the entered criteria as checked in the Step 2: Set and Finalize Stage Outcomes; Update Recommended Amounts section.

 

To select or unselect all participants, click the Check All link or Clear All link in the Step 1: Select Applications section.

 

The second step is to set the proposed and final stage outcomes for the selected participants.

 

To set the proposed stage outcomes for the review stage:

 

1.Follow the steps above to select the desired participant records.
2.Select the Proposed Stage Outcome.
3.Click the Set button. The page will refresh and display the selected value for the selected records in the Proposed Review Stage Outcome column.

 

Proposed stage outcomes are usually generated and then reviewed offline before setting the final stage outcomes. A list of records with the proposed stage outcome can be generated using the Query Tool.

 

To set the final stage outcomes for the review stage:

 

1.Follow the steps above to select the desired participant records.
2.Click the Finalize button. The page will refresh and display the a Yes or No in the Review Stage Outcome Finalized? (Y/N) column.

 

If this is not the final review stage, the next stage and/or cycle would begin after the stage outcomes had been finalized. If this is the final review stage, outcomes can be made.

 

The final step is to set the grantee status and outcome for the review cycle. This step is only taken in the final review cycle/stage. Once the status and outcome for the selected participants has been set, the participant will not longer be displayed on the page. The information can be edited from the Grants module.

 

To set the proposed status for the review cycle/stage:

 

1.Follow the steps above to select the desired participant records.
2.Select the Proposed Grant Status.
3.Click the Set button. The page will refresh and display the selected value for the selected records in the Proposed Grant Status column.

 

To set the proposed grant project outcome for the review cycle/stage:

 

1.Follow the steps above to select the desired participant records.
2.Select the Proposed Overall Grant Project Outcome.
3.Click the Set button. The page will refresh and display the selected value for the selected records in the Proposed Grant Project Outcome column.

 

To set the final the outcomes:

 

1.Follow the steps above to select the desired participant records.
2.Click the Finalize button in the Step 3: Set and Finalize Overall Grant Project Outcomes (Final Review Stage Only).

 

The grant status and/or outcome can be modified manually in the grant file.
