Copying Instructional Content

 

   

To copy the content:

 

1.Click on the Copy button next to the Task field in the Content Management section of the Content Manager page. The Content Manager Copy page is displayed.
2.Select the Program in the Copy To section. The page refreshes and displays the Funding Opportunity field.
3.Select the Funding Opportunity in the Copy To section. The page refreshes and displays the Funding Cycle field.
4.Select the Funding Cycle in the Copy To section. The page refreshes and displays the Task field.
5.Select the Task in the Copy To section. The page refreshes displaying the Save button.
6.Click on the Save button. Click on the OK button in the confirmation pop-up to continue the process.

 

Content can only be copied to the same task in another program, funding opportunity, and funding cycle selection. Content can not be copied to a different task with different pages, e.g., content can not be copied from the Submit Proposal task to a Submit Progress Report task.

 

7.To return to the Content Manager page without saving the changes, click on the Close button.
