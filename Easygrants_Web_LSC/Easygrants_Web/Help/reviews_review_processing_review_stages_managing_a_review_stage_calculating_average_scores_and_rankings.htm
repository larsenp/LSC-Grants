Calculating Average Scores and Rankings

 

   

The first step in determining final outcomes for a review stage is to calculate the average score and ranking for the participants.

 

To access the Review Stage Rankings page:

 

1.Click on the Reviews link in the left navigation.
2.Click on the Review Processing link. The Review Cycles page is displayed.
3.Click on the Manage button for a specific review cycle. The Participant/Committee Pool page is displayed.
4.Click on the Review Stages tab in the review cycle. The Review Stages page is displayed.
5.Click on the Manage button for a specific review stage. The Review Stage Assignments page is displayed.
6.Click on the Rankings tab in the review stage. The Review Stage Rankings page is displayed.

 

The Review Stage Rankings page displays all of the participants for the review cycle in the Participant  section with the following columns:

 

•Rank Order
•Average Score
•Participant
•Participant Organization

 

The results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

When first accessing the page, the Rank Order and Average Score columns will not have any values.

 

To calculate average scores for the participants:

 

1.Select the participants that need to be included by clicking on the checkbox next to the record.

 

To select or unselect all participants, click the Check All link or Clear All link.

 

2.Click on the Calculate/recalculate Average Scores Go button to generate an average score for each participant. The page refreshes and displays the “Last Run” information including the user the requested the average scores, the date/time, and the status.

 

When first running the average scores, the request will be put into to a queue. The status will be "Request is in queue to be executed".  To view the most recent status, click the refresh button for your browser.

 

3.The Average Scores will be displayed in the appropriate column. If no scores were submitted for a participant, the average score will be blank.

 

To calculate the ranking for the participants:

 

1.Select the participants that need to be included by clicking on the checkbox next to the record.

 

To select or unselect all participants, click the Check All link or Clear All link.

 

2.Click on the Calculate/recalculate Rank Order Go button to generate the ranking for each participant. The page refreshes and displays the “Last Run” information including the user the requested the rank order, the date/time, and the status.

 

When first running the average scores, the request will be put into to a queue. The status will be "Request is in queue to be executed".  To view the most recent status, click the refresh button for your browser.

 

3.The Ranking will be displayed in the appropriate column. If no scores were submitted for a participant, the ranking will be blank.

 

The ranking is based on the selected participants. The lower the score, the higher the rank.
