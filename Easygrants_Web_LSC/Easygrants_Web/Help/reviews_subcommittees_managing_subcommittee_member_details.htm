Managing Subcommittee Member Details

 

   

From the Subcommittee Members page you can view information on person contacts designated as subcommittee members and access editing features for specific reviewer subcommittee membership details. In addition, specific reviewers can be removed from the selected subcommittee. To access the Subcommittee Members page, click on the View button from the Committees page or the Save and Continue button from the Subcommittees Editor page.

 

The subcommittee members can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

To edit subcommittee member details:

       

1.Click on the Edit button next to the desired subcommittee member. The Committee Member Editor page is displayed.
2.Select the role of the Committee member (e.g., Member, Chair) (defaults to member).
3.Enter the Start and End Dates of the Committee member’s tenure of service.

 

   

The Start and End Dates for the committee member's tenure of service are for reporting purposes only.

 

4.Depending on whether or not the selected Committee member is an active participant on the committee, choose either yes or no in the Active? drop-down.
5.Click on the Save button to save the committee member details and remain on the Subcommittees Editor page.
6.Click on  the Save and Continue button to save the committee member details and return to the Subcommittees Members page.
7.To return to the Committees Member page without saving changes to the information, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display a page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To remove a subcommittee member:

 

1.Click on the Delete button next the desired committee member.
2.Click on the OK button in the pop-up confirmation message to continue with the deletion. The page refreshes and no longer displays the deleted member in the list.

 

The Action drop-down list in the Committee Members page contains options to manage a committee member’s information.

 

To access the Action drop-down list options:

 

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu are detailed in the following table.

 

Action Item

Description

Contact Info

The Contact Info features allow you to manage contact details, task activities, group memberships, uploaded documents, and

nominator information for the selected individual.

Login as this User

The Login as this User feature allows you to log in as the selected individual.

Send E-mail

The Send E-mail pages allow you to send custom or standardized e-mail messages to a selected individual or group through the system.

View Activity

The View Activity feature allows you to view that selected individual's activity.
