Creating Tasks

   

The Task Builder section on the Task Builder page allows the creation of task types other than eligibility such as application, internal review, or external review. Each task contains forms that are selected to make up the full task.

 

To add a new task:

 

1.Click on the Add button in the Task Builder section of the Task Builder page. The Task Builder Editor page will be displayed.
2.Enter the Task Name (required).
3.Enter the Description (required).
4.Select the Type (required).
5.Enter the Guideline File URL. If the URL is external to the Easygrants project, check the External URL box. If the URL is not external, the relative URL must be provided, e.g., /Implementation/Modules/Application/Help/<FileName>.html.
6.Check if it is appropriate for the task to be copied..
7.Select the Standard, Main, and Review and Submit forms to be included.

 

The Standard forms are used mainly in the public tasks such as Submit Application, Submit Progress Report, or Submit Final Report. The Main page provides detailed instructions and a summary of the requirements. The Review and Submit page provides details on the requirements, allows the user to view the full PDF, and, if all requirements have been met, the submission of the task.

 

8.Click on the Save button to save the information and remain on the Task Builder Editor page.
9.Click on the Save and Close button to save the information and return to the Task Builder page.
10.To return to the Task Builder page without saving the changes, click on the Close button.

 

To associate the forms with a task:

 

1.Click on the Add button in the Forms section of the Task Builder Editor page. The Task Builder Outcomes Editor page will be displayed. The Task Name will be displayed as read only.
2.Select the Form from the drop-down (required). The forms are created by the development team and associated as an available form in the a_STANDARD_FORMS table. Selecting the form will display the description.
3.Enter the Navigation Name (required). The navigation name will display in the left navigation as well as in the Validation sections on the Main and Review and Submit pages if being used.
4.Enter the Help File URL. If the URL is external to the Easygrants project, check the External URL box. If the URL is not external, the relative URL must be provided, e.g., /Implementation/Modules/Application/Help/<FileName>.html.
5.Enter the Order. The order will list the page in the left navigation based on the other order of the pages. If the Main and Review and Submit pages have been selected to be included, the Main page will always be the first page and the Review and Submit page will always be the last page.
6.Enter the Submission Validation.
7.Click on the Save button to save the information and remain on the Task Builder Form Editor page.
8.Click on the Save and Close button to save the information and return to the Task Builder Editor page.
9.To return to the Task Builder Editor page without saving the changes, click on the Close button.

 

To modify a selected task form, click on the Edit button for the desired record and follow the steps above. To delete the form from the task, click on the Delete button for the desired record.

 

To associate the outcomes with a task:

 

1.Click on the Add button in the Outcomes section of the Task Builder Editor page. The Task Builder Outcomes Editor page will be displayed. The Task Name will be displayed as read only.
2.Enter the Outcome (required).
3.Click on the Save button to save the information and remain on the Task Builder Outcomes Editor page.
4.Click on the Save and Close button to save the information and return to the Task Builder Editor page.
5.To return to the Task Builder Editor page without saving the changes, click on the Close button.

 

For tasks that will be reviewed by internal staff all possible outcomes should be added. For example, the application that is reviewed internally could include Approved for Committee Review.

 

To modify a selected task outcome, click on the Edit button for the desired record and follow the steps above. To delete the outcome from the task, click on the Delete button for the desired record.

 

To edit an existing task, click on the Edit button next to the desired record in the Task Builder section of the Task Builder page. The Task Builder Editor page is displayed with the selected record information. Use the steps above to complete the modification of the task.

 

To delete a task:

 

1.Click on the Delete button next to the desired record in the Task Builder section of the Task Builder page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

The Delete button is only available for tasks that have not been associated to a workflow.

 

To copy a task:

 

1.Click on the Copy button next to the desired record in the Task Builder section of the Task Builder page.
2.Click on the OK button in the confirmation pop-up to continue with the copy. The Copy Task page will be displayed.
3.Modify the New Task Name (required). The task name must be unique in order to finish copying the task.
4.Modify the Description (required).
5.Click on the Copy button to continue copying the task.
6.To return to the Task Builder page without saving the changes, click on the Close button.
