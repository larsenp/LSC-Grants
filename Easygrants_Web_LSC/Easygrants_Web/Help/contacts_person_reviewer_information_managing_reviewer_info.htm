Managing Reviewer Information

 

   

Person contacts can be designated as a reviewer in Easygrants making them available for searching in the Reviewers Easylist, assigning as a committee member, and assigning reviewer tasks to be completed (e.g., Submit Preliminary Review Scores).

 

If a person contact has not been designated as a reviewer, the Add button will be displayed in the Reviewer Information section of the Reviewer Info page. Reviewer designated contacts will have their key reviewer information displayed with an Edit button for managing the information.

 

To add a person contact as a reviewer:

 

1.Click on the Add button in the Reviewer Information section of the Reviewer Information page. The Reviewer Information Editor page is displayed.
2.Select if the person contact should be added to the reviewer list. Checking the box will make the person contact available in the Reviewers Easylist search results.
3.Select the Status (required).

 

Only person contacts designated as a reviewer with a status of "Active" will be available for committee membership.

 

4.Enter the Start Date (mm/dd/yyyy).
5.Enter the End Date (mm/dd/yyyy).

 

The reviewer start and end dates are for information purposes only.

 

6.Select the Area of Expertise.

 

•The list of area of expertise is managed from the Lookup Table Manager in the Administration module. The area of expertise can be used to help match reviewers to applications or grantee reports in Review Processing.
•To add an area of expertise, click on the desired value from the Available list box and click on the Add button. The value(s) will be added to the Selected list box.
•To add several values listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting values in a row or the <CTRL> key if selecting values throughout the list. Click on the Add button until all desired values have all been added to the Selected list box.
•To remove previously selected names from the Selected list box, select the value and click on the Remove button. The value(s) will be removed from the Selected list box.

 

7.Click on the Save button to save the information and remain on the Reviewer Information Editor page.
8.Click on the Save and Close button to save the information and return to the Reviewer Information page.
9.To return to the Reviewer Information page without saving the changes, click on the Close button.

 

To edit the reviewer information, click on the Edit button in the Reviewer Information section of the Reviewer Information page. The Reviewer Information Editor page is displayed with the previously entered information. Use the steps above to complete the modification.

 

To remove a person contact as a reviewer:

 

1.Click on the Edit button in the Reviewer Information section of the Reviewer Information page. The Reviewer Information Editor page is displayed.
2.Unselect that the person contact should be added to the reviewer list.
3.Update the Status (required).
4.Click on the Save button to save the information.
