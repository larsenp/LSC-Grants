Selecting the Committees to Include

 

   

To select the committees (reviewers) to include in the review cycle:

 

1.In the Committees to Include section, select the review committees that will be reviewed in the review cycle (required).

       

•The list of available committees is comprised of all review committees in Easygrants. Subcommittees are NOT included in the list.
•To add a committee, click on the committee name from the Available list box and click on the Add button. The committee(s) will be added to the Selected list box.
•To add several committees listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting committees in a row or the <CTRL> key if selecting committees throughout the list. Click on the Add button until all desired committees have all been added to the Selected list box.
•To remove previously selected committees from the Selected list box, select the committees and click on the Remove button. The committees(s) will be removed from the Selected list box.

 

To view the selected committee (reviewers) pool:

 

1.Click on the View of Selected Committee/Reviewer Pool link in the Committees to Include section of the Participant/Committee Pool page. A new window will open displaying the results.

 

       The results are displayed with the following columns:

               

•Reviewer
•Primary Organization
•Research Area
•Committee

 

The results can be sorted by the column headers (not including Research Area). Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.
