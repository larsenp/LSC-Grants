Managing Committees

 

   

To add a new committee:

       

1.Click on the Add button in the Committee Management section of the Committees page. The Committee Editor page is displayed.
2.Enter the Committee Name (required).
3.Select the Program from the drop-down.
4.Select the Time Period/Funding Cycle.
5.Depending on whether or not the committee is active, choose either yes or no in the Active Committee? drop-down (defaults to yes).

 

   

The Active Committee drop-down is for reporting purposes only.

 

6.In the Select Committee Members section, select each reviewer from the Available list. To populate the Available list, search for the reviewer by entering the reviewer's last name and clicking on the Find button.

               

•The list of available reviewers is comprised of all contacts in the system designated as reviewers. If you cannot locate the desired reviewer, either the contact record  has not been appropriately flagged as a reviewer record or the reviewer is not a contact in the system. First, check to see if the reviewer is a contact in the system using the Reviewers Easylist Search or the Contacts Easylist Search. Refer to the sections Searching for a Reviewer and Searching for a Contact for instructions on these search processes. If the reviewer is not a contact in the system, follow the instructions in the section Adding a Reviewer Contact Record.
•To add a reviewer name, click on the desired name from the Available list box and click on the Add button. The name(s) will be added to the Selected list box.
•To add several names listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting names in a row or the <CTRL> key if selecting names throughout the list. Click on the Add button until all desired names have all been added to the Selected list box.
•To remove previously selected names from the Selected list box, select the name and click on the Remove button. The name(s) will be removed from the Selected list box.

 

7.Click on the Save button after all selections have been made to save the committee details and remain on Committee Editor page.
8.Click on the Save and Continue button to save the committee details and continue with the addition of details for each committee member on the selected  committee. Refer to the section Managing Committee Member Details for additional information.
9.To return to the Committees page without saving changes to the information, click on the Close button.

 

Saving the page without entering required information or incorrect with formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a committee, click on the Edit button next to the desired record in the Committee Management section of the Committees page. The Committee Editor page is displayed with the selected record information. Use the steps above to complete the modification of the committee.

 

To delete a committee:

 

1.Click on the Delete button to the right of the desired record in the Committee Management section of the Committees page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

Users will only be able to delete committees where there are no review task assignments for the committee members.
