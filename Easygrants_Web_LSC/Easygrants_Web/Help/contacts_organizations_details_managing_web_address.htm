Managing Organization Web Address Info

 

   

Web address information can be tracked at the organization level. There can only be one primary Web address for an organization.

 

To add an organization Web address record:

 

1.Click on the Add button in the Web Address section of the Organization Details page. The Web Address Editor page is displayed.
2.Select if this is the primary Web address for the organization.
3.Select the Type (required).
4.Enter the Web address (required).
5.Click on the Save button to save the information and remain on the Web Address Editor page.
6.Click on the Save and Close button to save the information and return to the Organization Details page.
7.To return to  the Organization Details page without saving changes, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit an organization Web address record, click on the Edit button next to the desired record in the Web Address section of the Organization Details page. The Web Address Editor page is displayed with the selected record information. Use the steps above to complete the modification of the Web address.

 

To delete an organization Web address record:

 

1.Click on the Delete button to the right of the desired record in the Web Address section of the Organization Details page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
