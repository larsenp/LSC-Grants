Task Assignment History

 

   

The Task Assignment History section of the Grant Activity page displays all workflow task assignments associated with the grant record. The records will be displayed in ascending order by the assigned person's last name and then first name with the following columns:

 

•Action
•Assigned Person
•Task Name
•Task Status
•Task Outcome
•Submit Date

 

The records can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

The records are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

To access the Action drop-down list options:

 

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu are detailed in the following table.

 

Action Item

Description

Tasks – Edit this Task

The Tasks – Edit this Task feature allows you to edit the details of a task assignment, such as the status, outcome, and task dates.

Login as this User

The Login as this User feature allows you to log in as the related primary contact (applicant).

Contact Info

The Contact Info features allow you to manage contact details, task activities, uploaded documents, and reviewer information for the related primary contact (applicant).

Tasks – Add New Task

The Tasks – Add New Task feature brings you to the Assign Task Editor where you can view the current task and add a new task for the selected individual.

View PDF

The View PDF feature allows you to view the PDF file associated with the grant.
