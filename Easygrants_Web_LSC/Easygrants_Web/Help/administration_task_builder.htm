Task Builder

 

   

The Task Builder section of the Administration module allows the creation of tasks that will be completed by Easygrants users, including public users (applicants, reviewers, grantees) and internal staff.

 

The Task Builder has been broken out into two sections allowing:

 

•The creation of the eligibility tasks.
•The creation of other task types including, application, internal review, external review, grant, and grantee reporting.

 

The tasks created in the task builder can then be used in the Workflow Builder to create the grant lifecycle tasks that should be completed for a particular Funding Cycle.

 

To access the Task Builder:

 

1.Click on the Administration link in the left navigation.
2.Click on the Task link in the left navigation. The Task Builder page will be displayed.
