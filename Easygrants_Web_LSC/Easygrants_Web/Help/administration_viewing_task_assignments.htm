Viewing Task Assignments

 

   

To display task assignments for a specific Funding Cycle:

 

1.Select the Program. The page will refresh and display the related Funding Opportunities in the Funding Opportunity drop-down.
2.Select a Funding Opportunity. The page will refresh and display the related Funding Cycles in the Funding Cycle drop-down.
3.Select a Funding Cycle. The page will refresh and display the related Tasks in the Task drop-down.
4.Select a Task. The results will be displayed in the Task Assignments section.

 

The columns that are displayed in the Search Results include:

 

•Action
•Assigned Person
•Grantee Project Person
•Status
•Outcome
•Submitted Date

 

The search results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Search results are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

To access the Action drop-down list options:

 

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu for task assignments are detailed in the following table.

 

Action Item

Description

Contact Info

The Contact Info features allow you to manage contact details, task activities, group memberships, uploaded documents, and nominator information for the selected individual.

Login as this User

The Login as this User feature allows you to log in as the selected individual.

Send E-mail

The Send E-mail pages allow you to send custom or standardized e-mail messages to a selected individual or group through the system.

Tasks - Add New Task

The Tasks - Add New Task feature brings you to the Assign Task Editor where you can view the current task and add a new task for the selected individual.

Tasks - Edit this Task

The Tasks - Edit this Task feature allows you to edit the details of a task assignment, such as the status, outcome, and task dates.

View Activity

The View Activity feature brings you to the Person Contact - Activity page and allows you to view and edit all activities (both grant and application) for the contact selected.

View Application PDF

The View Application PDF action allows you to view the related PDF file for the selected task. If there is no related PDF file for the selected task, the system will display the message "Required PDF is not available."

View Project Activity

The Project Activity feature allows you to view and edit Grant information. The Grant Activity page is displayed.

View Project Outcomes

The View Project Outcomes feature allows you to edit grantee project task outcomes for a specific Funding Cycle.
