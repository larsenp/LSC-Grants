Eligibility Workflow Builder

 

   

The Eligibility Workflow Builder allows the user to select the eligibility task to be completed as part of the workflow. Eligibility includes a series of yes or no questions that are answered by the potential grantees to help determine if they are eligible to apply for a particular funding cycle.

 

Eligibility quizzes are set up using the Eligibility Builder. Refer to the Eligibility Builder section for more details.

 

To access the Eligibility Workflow Builder page:

 

1.Click on the View button next to the desired Funding Cycle in the Funding Cycle Management section of the page.
2.The page refreshes and displays the workflow builder tabs for each grant lifecycle phase starting with the Eligibility Workflow Builder.
3.Click on the Eligibility tab.

 

The Eligibility Workflow Builder is comprised of the following steps:

 

1.Selecting the eligibility task to be completed by the potential grantees.
2.Creating the eligibility quiz associated with the task.
3.Determining the workflow based on the outcome of the eligibility task.
