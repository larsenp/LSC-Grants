Managing Budgets

 

   

From the Budgets section of the Grant Financials page, staff can add or modify budgets associated with the grant either entered directly in the grant file or by the applicant/grantee during the application or grantee reporting phase of the grant lifecycle. This feature gives you the ability to save and differentiate between multiple versions of a budget (e.g., Application, Awarded, Expenditure).

 

To add a new version of a budget submitted by an applicant:

 

1.Click on the Add button in the Budgets section of the Grant Details page. The Budget Editor page is displayed.
2.Select the Budget Version (required)
3.Enter the Description (required).
4.Enter the line item costs for the all applicable budget years.
5.Enter Notes as needed. Click on the Notes next to the line item. The Budget Item Notes Editor page is displayed.
a)Click on the Save button to save the information and remain on the Budget Item Notes Editor page.
b)Click on Save and Close to save the information and return to the Budget Editor page.
c)To return to the Budget Editor page without saving the changes, click on the Close button.
5.Click on the Calculate and Save button to save the data, calculate the sub-totals and totals for the budget line items, and remain on the Budget Editor page.
6.Click on the Save and Close button to save the data and return to the Grant Financials page.
7.To return to the Grant Financials page without saving the changes, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display a page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a budget, click on the Edit button next to the desired record in the Budgets section of the Grant Financials page. The Budget Editor page is displayed with the selected record information. Use the steps above to complete the modification of the budget.

 

Budgets can be copied and then modified in the Budgets section of the Grant Financials page. For example, an application budget can be copied and then modified to create the awarded budget. The awarded budget would be the budget the primary contact/PI will report on in the expenditures report.

 

To copy a budget:

 

1.Click on the Copy button next to the desired record in the Budgets section of the Grant Financials page. A pop-up message is        displayed.
2.Click on the OK button in the pop-up confirmation message to continue with the copy. The page refreshes and displays the new budget with "Copy of XXX" where "XXX" is the description of the original budget version.

 

To delete a budget:

 

1.Click on the Delete button to the right of the desired record in the Budgets section of the Grant Details page.
2.Click on the OK button in the pop-up confirmation message to continue with the deletion. The page refreshes and no longer displays the deleted budget in the list.
