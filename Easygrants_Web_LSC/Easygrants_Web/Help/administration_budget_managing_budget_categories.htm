Managing Budget Categories

 

   

To search for the appropriate funding cycle:

 

1.Select a Program Name from the drop-down list. The page refreshes with the associated Funding Opportunities.
2.Select a Funding Opportunity from the drop-down list. The page refreshes with the associated Funding Cycles.
3.Select the Funding Cycle Name.

 

If at least one budget category already exists for the selected funding cycle, the page refreshes with the associated budget categories in the Budget section of the Budget page.

 

To add a new budget category:

 

1.Click Add in the Budget section of the Budget page. The Budget Editor displays.
2.Enter the Budget Category Name (required).
3.Enter the Sort Order (required).
4.Click Save to save your changes and stay on the Budget Editor page.
5.Click Save and Close to save your changes and return to the Budget page.
6.Click Close to return to the Budget page without saving your changes.

 

To edit a budget category, click on the Edit button next to the desired record in the Budget section of the Budget page. The Budget Editor page is displayed with the selected record information. Use the steps above to complete the modification of the budget.

 

To delete a budget category:

 

1.Click on the Delete button to the right of the desired record in the Budget section of the Budget page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

The Delete button is only available for budget categories where the line items do not have any values associated with them.
