Accessing Workgroup Tasks

 

   

To perform a particular action on a workgroup task:

       

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu for the Workgroup Tasks are detailed in the following table.

 

Action Item

Description

Transfer to Personal Tasks

The Transfer to Personal Tasks feature allows you to send a workgroup task to the personal task list for completion/submission of the task. Once a task has been transferred it will be displayed in your Personal Tasks section.

View Project Outcomes

The View Project Outcomes feature allows you to edit intermediate Funding Cycle outcomes and to select a final grantee Funding Cycle outcome.

View Project Tasks

The View Project Tasks feature allows you to view detailed information on related assignments.

View PDF

The View PDF feature allows you to view the related PDF file for the selected task. If there is no related PDF file for the selected task, the system will display the message "Required PDF is not available."

 

 

Once the task has been transferred, it will no longer be available for other users in the workgroup. To reassign the task to another user, use the Reassign link in the Personal Tasks section.
