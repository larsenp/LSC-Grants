Notifying Review Committee Members

 

   

As the review stage opens, the review committee can be notified using the Review Notification page.

 

To send the e-mail notification to review committee members:

 

1.Click on the Notification tab for the Review Stage. The Review Notification page is displayed.
2.Select the committees to receive e-mail notification from the Identify Committees to Notify section.

 

•The list of committees is comprised of all committees selected for the review cycle.
•To add a committee for notification, select the desired name from the Available list box and click on the Add button. The name will be added to the Selected list box.
•To add several names listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting names in a row or the <CTRL> key if selecting names throughout the list. Click on the Add button until all desired names have all been added to the Selected list box.
•To remove previously selected names from the Selected list box, select the name and click on the Remove button. The name will be removed from the Selected list box.

 

3.Click on the Save and Continue button to continue to send the notification. The Send E-mail page is displayed.

 

The Send E-mail page will display the first five names on the recipient list with the total count of recipients. A separate e-mail will be sent to each recipient.

 

4.Select the E-mail Template as needed. Selecting an e-mail template will pre-populate the information based on the selected template.
5.Enter or modify the From name (required).
6.Enter or modify the CC e-mail address.
7.Enter or modify the BCC e-mail address.

 

An e-mail will be sent to the CC and BCC e-mail address for each recipient on the list. For example, if you are sending five e-mail, five e-mails will be sent to each address in the CC and BCC fields.

 

8.Enter or modify the Subject (required).
9.Select or modify the E-mail Format. Clicking the checkbox will allow HTML formatted e-mails.
10. Select of modify the E-mail History. Clicking the checkbox will log the e-mail and display it in the Person Contact Correspondence tab for the recipients.
11. Enter or modify the Text (required).
12.Click on the Send button to send the e-mail notification to all selected committee. The Send E-mail page will display a confirmation message that the e-mail was sent. To return to the Review Stage Notification page after an e-mail is sent, click on the Close button.
13.Click on the Cancel button to cancel the e-mail notification and return to the Review Stage Notification page.

 

The Reviewer Notification page will display the history of the notifications in the E-mail Notification History - For This Review Cycle section.

 

To send a test e-mail prior to sending the notification, on the Send E-mail page in the Send Test E-mail section, enter a test e-mail with a valid e-mail address and click on the Send Test E-mail button.
