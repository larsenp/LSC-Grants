Relating Uploads to Tasks

 

   

Upload requirements can be associated with a particular task in a funding cycle workflow through the Upload Task page. Associating an upload will display that upload type on the Uploads page of the selected task, e.g., Submit Proposal. This will make up the full list of upload types available for the task assignment. Not all upload types have to be required.

 

To select the task to associate uploads to:

 

1.Select the Program (required). The page will refresh and display the Funding Opportunity drop-down.
2.Select the Funding Opportunity (required). The page will refresh and display the Funding Cycle drop-down.
3.Select the Funding Cycle (required). The page will refresh and display the Task drop-down.
4.Select the Task (required). The page will refresh and display Uploads section on the page. If previous upload associations have been made, those will display in the section.

       

To associate an upload:

 

1.Click on the Add button in the Uploads section of the Upload Task page. The Upload Task page is displayed.
2.Select the Upload Name (required).
3.Uncheck Include in Task as needed.

 

The "Include in Task" field will default to checked. Removing the check will remove the upload from the available type list in the selected task. This should be unchecked if the change is made after submissions have been completed in order to preserve historical data.

 

4.Select if the upload is included in the pdf.
5.Select if the upload is Required.
6.Select if the upload is part of the general upload.
7.Select the Organization Eligibility that overrides the requirement for the upload.

 

•The list of available organization eligibility is comprised of all values in the c_ORGANIZATION_ELIGIBILITY table. Refer to the sections Managing Lookup Table Values for instructions on adding new values.
•To add a value, click on the desired record from the Available list box and click on the Add button. The value(s) will be added to the Selected list box.
•To add several values listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting values in a row or the <CTRL> key if selecting values throughout the list. Click on the Add button until all desired records have all been added to the Selected list box.
•To remove previously selected values from the Selected list box, select the value and click on the Remove button. The value(s) will be removed from the Selected list box.

 

8.Click on the Save button to save the information and remain on the Upload Task page.
9.Click on the Save and Close button to save the information and return to the Upload Task page.
10.To return to the Upload Task page without saving the changes, click on the Close button.

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit the associated upload, click on the Edit button next to the desired record in the Uploads section of the Upload Task page. The Upload Task page is displayed with the selected record information. Use the steps above to complete the modification of the association.
