Administration

 

   

The Administration section of the Easygrants Manager includes the tools for managing the setup of the Easygrants system. This includes:

 

•Configuration:  Provides the functionality to setup and maintain Programs, Funding Opportunities, and Funding Cycles including the associated workflow, eligibility quiz requirements, and the task submissions.
•Budget: Provides the functionality to set up and maintain budgets for specific funding cycles.
•Content Manager: Provides the functionality to setup and maintain the page instructional content for tasks at the Funding Cycle level.
•Eligibility: Provides the functionality to setup and maintain eligibility quizzes at the Funding Cycle level.
•Task: Provides the functionality to set up and maintain Easygrants tasks.
•Upload Manager: Provides the functionality to manage upload types including the association of templates to upload requirements, e.g., format, page count, etc.

 

To access the Administration section main page, click on the Administration link in the left navigation. The Program Management page is displayed.
