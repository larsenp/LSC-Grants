Viewing Your Grants Portfolio

 

   

To view your grants portfolio:

 

1.Click on the Go to My Grants Portfolio link in the Options section of the Home page. The My Grants Portfolio page is displayed.

 

To filter through your grants:

       

1.Select one or more search criteria contained in the Filter My Grants section of the My Grants Portfolio page.

 

       The search filter criteria configured for the Easygrants system are:

 

•Grant Status
•Funding Opportunity

 

   

By default, the Grants Portfolio will contain all grants for which the logged in user is the primary staff member affiliated to the grant .

 

2.Click on the Find button to execute the filter. The page refreshes and displays a grant(s) matching the selected criteria.

 

Each grant is displayed in a separate section listed with the Easygrants ID and the following information:

 

•Easygrants ID
•Legacy Grant ID
•Grantee Organization
•Primary Contact/PI
•Status
•Program
•Outcome
•Funding Opportunity
•Funding Opportunity Cycle
•Project Title

 

   

To go to the Grantee Organization's organization record, click on the Grantee Organization name link. To go to the Primary Contact/PI's person record, click on the Primary Contact/PI name link.

 

3.Click the Grant Details button to go to the selected grant file to view more details on the grant.
4.Click on the Clear button to clear any entered filter criteria.
5.To return to the Home page, click on the Close button.
