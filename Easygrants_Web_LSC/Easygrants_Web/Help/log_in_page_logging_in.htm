Logging In

 

   

To log in to the Easygrants Manager:

 

1.Open the Internet Explorer Web browser (Internet Explorer 6.0 or higher is required for the Easygrants Manager features of the system).
2.Navigate to the Easygrants URL in your Web browser. The Easygrants system Log In Page is displayed.
3.Enter your Login ID and Password into the fields provided. The Login ID is your primary e-mail address on record.
4.Click on the Log In button. Once logged in, your Easygrants Manager Home page is displayed.

 

If you enter incorrect login information, a message will appear on the page directing you to try your entries again. If you have forgotten your login information, refer to Forgot Password below.

Trying to log in with a valid login ID and invalid password more than three times will lock the account. To unlock the account, contact the system administrator.
