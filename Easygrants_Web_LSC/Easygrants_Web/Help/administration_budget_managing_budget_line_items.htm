Managing Budget Line Items

 

   

To search for the appropriate budget category:

 

1.Select a Program Name from the drop-down list. The page refreshes with the associated Funding Opportunities.
2.Select a Funding Opportunity from the drop-down list. The page refreshes with the associated Funding Cycles.
3.Select the Funding Cycle Name. The page refreshes with the budget categories that have been created for the Funding Cycle.

 

The columns that are displayed in the Budget section of the page include:

 

•Budget Category
•Line Items

 

To manage budget line items, click on the Edit button next to the desired record in the Budget section of the Budget page. The Budget Editor page is displayed with the selected record information.

 

To add a new budget line item,

 

1.Click Add in the Add or Edit Budget Line Items section of the Budget Editor page. The Budget Line Item Editor page displays.
2.The Program, Funding Opportunity, Funding Cycle and Budget Category Name display as read-only.
3.Enter the Line Item Name (required).
4.Enter the Sort Order (required).
5.Click Allow Notes if notes can be associated with the Line Item.
6.Click Save to save your changes and stay on the Budget Line Item Editor page.
7.Click Save and Close to save your changes and return the Budget Editor page.
8.Click Close to return to the Budget Editor page without saving your changes.

 

To edit a budget line item, click on the Edit button next to the desired record in the Add or Edit Budget Line Items section of the Budget Editor page. The Budget Line Item Editor page displays. Use the steps above to complete the modification to the budget line item.

 

To delete a budget line item:

 

1.Click on the Delete button to the right of the desired record in the Add or Edit Budget Line Items section of the Budget Editor page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

The Delete button is only available for budget categories where the line items do not have any values associated with them.
