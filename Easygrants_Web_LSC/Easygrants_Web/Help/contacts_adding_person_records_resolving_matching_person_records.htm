Resolving Matching Person Records

 

   

Upon saving the person information, Easygrants checks for existing contact records based on the following:

 

•An exact match between the first and last name
•An exact match between the first names and the new contact middle name and an existing contact last name
•An exact match on the e-mail address

 

If an exact match is found, the user is redirected to the Matching Persons page with the matching records displayed. To view the details for a matching record, click on the Name link. The Matching Person Contact Details page is displayed in a new window providing a read only view of the full contact information for the person record.

 

To continue adding the person record, click on the Create a New Contact button at the bottom of the page. To return to the Add Person Contact page without creating the contact, click on the Close button.

 

The Create a New Contact button will not be available if there is a duplicate match on the e-mail address. E-mail addresses must be unique in Easygrants.
