Adding a Grant

 

   

To add a new grant:

 

1.Click on the Add Grant button in the Add Grant Record section of the Grants Easylist page. The Add Grant page is displayed.
2.Select the Program Name. The page will refresh to display the related Funding Opportunity drop-down.
3.Select the Funding Opportunity. The page will refresh to display the related Funding Cycle crop down.
4.Select the Funding Cycle Name. The remaining data fields for completing the grant record are displayed.
5.Select the Primary Staff contact (required). The selector is filtered to only allow the selection of staff or administrative users in Easygrants.
6.Select the Grantee Organization contact (required).
7.Select the Primary Contact/PI (required).

 

To select the contacts, click on the Select button next to the field. The appropriate selector is displayed - Person or Organization Selector. Follow the steps to complete the selection process.

 

8.Enter the Project Title.
9.Enter the Project Description.
10.Enter the Grantee Project Term (Start and End dates).
11.Enter the Grant Requested Amount.
12.Enter the Award Amount.
13.Enter the Award Date.
14.Select the Status (required).
15.Select the Outcome.
16.Click on the Save and Continue button to save the information and go to the Grant Details page for the newly added grant.
17.To return to the Grants Easylist page without saving the information, click on the Close button.
