Adding Person Records

 

   

To add the person contact:

       

1.Click on the Add button in the Add Contact Record section on the Contact Easylist page. The Add Person page is displayed.
2.Select the Prefix.
3.Enter the First Name (required).
4.Enter the Middle Name.
5.Enter the Last Name (required).
6.Select the Suffix.
7.Enter the E-mail Address (required).

 

   

E-mail addresses in Easygrants must be unique. If the e-mail address being entered already exists in the system, a validation message will be displayed. Click on the Close button and return to the Contacts Easylist to search by the e-mail address.

 

8.Select if the contact should not be contacted.
9.Select if the contact is deceased.
10.Click on the Save button to save the information, creating the person record and remain on the Add Person page.
11.Click on the Save and Close button to save the information and return the Contact Easylist page.
12.To return to the Contact Easylist page without saving the changes, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.
