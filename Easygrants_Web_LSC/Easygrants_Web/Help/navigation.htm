Navigating the Easygrants Manager

 

   

Within the Easygrants Manager, staff and administrative users have access to tools that allow you to manage application and review processes; manage grant programs and grantee reports; manage contacts; manage and set-up Funding Opportunities, Funding Cycles and tasks; manage e-mail communication; and, utilize reporting functions.

 

The following table details the sections within the Easygrants Manager.

 

Link

Description

Home

The Home section provides access to assigned personal and workgroup tasks as well as the ability to view the grant portfolio and manage the password information.

Applications

The Applications section provides the tools to search for and manage applications.

Reviews

The Reviews section provides the tools to search for reviewers and review assignments and, to manage committees and the external review processes.

Grants

The Grants section provides the tools to search for grant files and to manage grant information, including the payment and reporting schedules.

Grantee Reports

The Grantee Reports section provides the tools to search for and manage grantee reports.

Contacts

The Contacts section provides the tools to add person and organization contacts and edit existing contact information.

Contact Categories

The Contact Categories section provides the tools to add new categories used to group person contacts.

E-mail Templates

The E-mail Templates section provides the tools to set up standard e-mails used in the workflow, for reminders, and from the send e-mail functionality in the action list and query tool.

Reminders

The Reminders section provides the tools to set up e-mail communications related to assigned tasks based on dates.

Task Assignments

The Task Assignment Section provides the ability to view and manage all  assignments for a specific task under a Program, Funding Opportunity and Funding Cycle.

Reports

The Reports section provides access to a query tool where you can create and access ad-hoc reports.

Administration

The Administration section provides the tools for managing the Easygrants system including the Programs, Funding Opportunities and Funding Cycles and their related workflow, eligibility quizzes, tasks, workgroups, task uploads, and lookup table values. Important: Only users of the Administrative type will see the link for administration. Making changes in this section will impact how the system operates.

FAQ

The FAQ section provides you with answers to frequently asked questions regarding the Easygrants application.

Help

The Help link provides access to detailed instructions and additional information for each section of the system.

Log Out

The Log Out link is used to exit the system.
