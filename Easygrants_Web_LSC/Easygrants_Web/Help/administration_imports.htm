Import

 

   

The Import functionality allows users to import specific data in Excel format to certain sections of Easygrants system . The import process consists of importing the file in the designated format, reviewing and committing to the import, and listing the historical requests.

 

Currently Easygrants is only configured to import payment information. The import must be in the specific file format and the payment ID must be included in the import data.

       

To access the Import Management functionality, click on the Administration link in the left navigation. The Program Management page is displayed. Click Import. The Import Management page is displayed.
