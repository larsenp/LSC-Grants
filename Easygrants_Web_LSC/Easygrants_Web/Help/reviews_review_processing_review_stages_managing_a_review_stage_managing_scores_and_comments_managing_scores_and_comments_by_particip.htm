Managing Scores and Comments by Participant

   

Review scores and comments can be viewed and updated by participant within the View and Modify Conflicts/Preferences, Scores, and Comments section of the Review Stage Scores and Comments page of the review stage.

 

To view the assignments and submission status for a participant:

 

1.Select the Participant name next to the By Participant field.
2.Click on the Go button. The Scores and Comments by Participant page is displayed.

 

The Scores and Comments by Participant page displays the selected participant information and the reviewers assigned in the View and Modify Conflicts/Preferences, Scores, and Comments section with the following columns:

 

•Reviewer
•Assignment Type
•Status
•Conflict/Preference
•Score
•Options

 

The results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

In the View and Modify Conflicts/Preferences, Scores, and Comments section, conflicts and preferences and submission results can be modified.

 

To return to the Review Stage Scores and Comments page for the review stage, click on the Close button.

 

To edit a single review score or comment for the participant:

 

1.Click on the Edit button next to the desired review assignment record. The Scores and Comments Editor page is displayed. The review assignment information will be displayed as read only.
2.Modify the Conflict and Preference selecting the Conflict/Preference and the Conflict Type (Financial, Organizational, Personal).
3.Modify the Score.
4.Modify the Reviewer Comments.
5.To view the Reviewer Comments Upload, click the View PDF button.

 

If no PDF is available, the system will display a message that the requested PDF is not available. This will either be displayed if not PDF was uploaded by the reviewer or the task does not require an upload.

 

6.Modify the Staff Edited Comments.

 

Staff Edited Comments are used to provide feedback to the participant.

 

7.Click on the Save button to save the information and remain on the Scores and Comments Editor page.
8.Click on the Save and Close button to save the information and return to the Scores and Comments by Participant page.
9.To return to the Scores and Comments by Participant page without saving changes to the information, click on the Close button.
