Viewing Person Correspondence

 

   

To view the correspondence e-mail details:

 

1.Click on the View button next to the desired record in the Correspondence E-mails section of the Correspondence page. The View Correspondence page is displayed.
2.To resend the message to the address in the To field, click the Resend button. A pop-up message is displayed. Clicking on the OK button will close the pop-up and continue to resend the message. The latest resend date will be updated to the current date. To cancel resending the message, click the Cancel  button.
3.To return to the Correspondence page, click on the Close button.

 

       

HTML formatted e-mails will display in the View Correspondence page with the HTML tags. For example, if the word bold was included in bold text it will display as <b>Bold</b>.
