Review Grantee Submission

 

   

The Review Grantee Submission page allows the user to review a resubmission request before the request notice is sent to the Applicant/Grantee.

 

Specific instructions related to resubmission will be displayed in the Instructions section of the Review Grantee Submission page.

 

To submit the resubmission request:

 

1.Review the Grant Information.
2.Review the email information. Users may add additional recipients (i.e. cc, bcc), modify the subject line and/or modify the actual text of the email as needed.
3.Click Save to save the data entered and stay on this page. The email is not sent.
4.Click Save and Send to save all of the data entered and send the email to all target recipients.  Click Cancel to make additional pages to the page. Click OK to send the email.

 

Clicking OK to send the email will also update the task status to Complete and reopen it with a status of Resubmit and an Outcome of Incomplete. The user will be redirected to the home page.

 

5.Click Close to navigate back to Resubmission page without saving any changes made to the page..
