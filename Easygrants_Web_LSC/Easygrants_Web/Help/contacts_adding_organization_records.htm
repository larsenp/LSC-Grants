Adding Organization Records

 

   

To add an organization:

       

1.Select Organization from the Add Contact Record section on the Contact Easylist page. Click the Add button. The Add Organization Contact page is displayed.
2.Enter the Legal Organization Name (required).
3.Enter the Common Organization Name (required).
4.Select the Type.
5.Enter the Employer Identification Number (EIN).
6.Enter the Address Line 1 (required), Address Line 2 and Address Line 3.
7.Enter the City (required).
8.Select the State/Province (required).
9.Enter the Region.
10.Enter the Zip/Postal Code (required).
11.Select the Country (required).
12.Enter the Phone Number.
13.Enter the Web site Address.
14.Select if the organization is available for public selection. This will make the organization available in the Organization Selector used by public users.
15.Select the Eligibility Status.
16.Click on the Save button to save the information, create the organization record, and go to the Edit Organization Contact page.
17.Click on the Save and Continue button to save the information, create the organization record, and go to the Organization Details page.
18.To return to the Contacts Easylist page without saving the information, click on the Close button.

 

   

The address, phone, and Web address entered will be marked as the primary information for the Organization record. This can be modified from the Organization Details page.

 

Saving the page without entering required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.
