Label Manager

 

   

The Label Manager feature allows the administrator to enter a Preferred Label Name to display for a select number of fields throughout the system.

 

All fields that can be configured with a preferred name are displayed. The labels can be sorted by the column headers Default Label Name and Preferred Label Name. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

To enter a preferred label name:

 

1.Click on the Edit button in the Label Manager section of the Label Manager page. The Label Manager Editor page is displayed.
2.Enter the Preferred Label Name (optional).
3.Click on the Save button to save the information and remain on the Label Manager Editor page.
4.Click on the Save and Close button to save the information and return to the Label Manager page.
5.To return to the Label Manager page without saving the changes, click on the Close button.
