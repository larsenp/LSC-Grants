Maintenance Alert Management

 

   

To Manage the Maintenance Alert Messages:

 

In the Maintenance Alerts Page:

 

1.Enter the desired plain text and HTML content in the Alert Message section.
2.Clicking Preview button will save the message in the database and open Maintenance Alert Preview page and the content is displayed.
3.Clicking Close button in the Maintenance Alert Preview page will close the Maintenance Alert Preview page.

 

In Display Message Section:

 

1.Select Yes or No. Yes will display the maintenance page only on the Easygrants Log In page.
2.Select the desired start and end dates and times to display the message. Calendar control can be used to select the dates and drop-down lists can be used to pick the times.

 

All times will be displayed in EST zone.

 

In Disable Login Section:

 

1.Select Yes or No. Yes will not allow the users to login to the Easygrants system while the maintenance alert is displayed on the Log In Page.
2.Select the disabled login start date and time to disable the login while the maintenance alert is displayed if you select Yes in the previous step. Calendar control can be used to select the dates and drop-down lists can be used to pick the times.

 

All times will be displayed in EST zone.

 

3.Click on the Save button to save the content and selection in the Maintenance Alert Page.
