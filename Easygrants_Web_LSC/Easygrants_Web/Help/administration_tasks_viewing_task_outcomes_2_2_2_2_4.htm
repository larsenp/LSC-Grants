Selecting the Eligibility Task

 

   

To select the task to be included in the eligibility workflow:

 

1.Click on the Add button in the Eligibility Task Information section of the Eligibility Workflow Builder page. The Workflow Builder - Task Editor page is displayed.
2.Select the Task (required). The task drop-down consists of all tasks with a type of Eligibility.
3.Enter the Description (required). The description will be displayed on the Home page.
4.Modify the Default Status as needed. The default status will be Incomplete.
5.Select the Assign To (required). The Assign To drop-down consists of the public user, the primary staff affiliated with the grantee project, or the existing workgroups in Easygrants.
6.Enter the task dates, Start, Open, Due, Close, and End (required).

 

•Start Date: The date that the task will be displayed on the Home page. The task will not be enabled.
•Open Date: The date that the task is available to access.
•Due Date: The date that is published as the deadline for the task submission.
•Close Date: The date that the task is no longer available to access. It still displays on the Home page.
•End Date: The date the task no longer displays on the Home page.

 

7.Check if this is the initial task in the workflow. The initial task is the starting point for the grant lifecycle. There can only be one initial task for the workflow.
8.Click on the Save button to save the information and remain on the Workflow Builder - Task Editor page.
9.Click on the Save and Close button to save the information and return to the Eligibility Workflow Builder page.
10.To return to the Eligibility Workflow Builder page without saving the changes, click on the Close button.

 

Once the Eligibility task has been added, the Add button will no longer be displayed.

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit the task to be included in the eligibility workflow, click on the Edit button next to the desired record in the Eligibility Task section of the Eligibility Workflow Builder page. The Workflow Builder - Task Editor page is displayed with the selected record information. Use the steps above to complete the modification of the task.

 

To delete the eligibility task to be included in the workflow:

1.Click on the Delete button next to the desired record in the Eligibility Task section of the Eligibility Workflow Builder page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

The Delete button is only available for records that do not have active task assignments and are not being used in the workflow.
