Searching for a Grant (Advanced Search)

 

   

To search for a grant using the advanced search:

       

1.Enter or select one or more search criteria contained in the Easylist Search section of the Grants Easylist page.

 

       The search filter criteria configured for the Easygrants system are:

 

•Easygrants ID
•Legacy Grant ID
•Grant Status
•Grant Outcome
•Grantee Organization
•Last Name - Primary Contact/PI
•First Name - Primary Contact/PI
•E-mail - Primary Contact/PI
•Project Title
•Time Period / Award Cycle
•Primary Staff Last Name
•Primary Staff First Name
•Program Name
•Funding Opportunity
•Funding Cycle Name

 

   

The Easylist functionality will assume an "And" for all the criteria. The more information you enter, the more the search results will be narrowed.

   

The Easylist functionality will assume a "contains" for text entry criteria and an "equals" for drop-down criteria.

 

2.Click on the Find button to execute the search. The page refreshes and displays a Search Results section containing the records matching the search.

 

       The columns that are displayed in the Search Results include:

 

•Action
•Easygrants ID
•Grantee Organization
•Program Name
•Funding Cycle Name
•Primary Contact/PI
•Status
•Legacy Grant ID

 

The search results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Search results are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

3.Click on the Easylist Search section header to open the Search section again.
4.Click on the Clear button to clear any entered search criteria and start a new search.
5.Click on the Quick Search link to return to the quick search fields.

 

To access the Action drop-down list options:

 

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu for the Grants Easylist are detailed in the following table.

 

Action Item

Description

Grant Info

The Grant Info feature allows you to manage the details of a grant, including people affiliations, budget information, progress report information, and payment details.

Contact Info

The Contact Info features allow you to manage contact details, task activities, group memberships, and uploaded documents for the selected individual.

View Project Activity

The View Project Activity feature will redirect the user to the Activity tab of the selected Grant File.

View Project Outcomes

The View Project Outcomes feature allows you to edit intermediate Project outcomes and to select a final grantee Project outcome.
