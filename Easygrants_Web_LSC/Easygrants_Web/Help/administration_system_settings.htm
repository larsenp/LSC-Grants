System Settings

 

   

The System Settings section allows the administrator to manage features that affect the system and are used throughout Easygrants.

 

The features available are:

 

•Label Manager
•Event Manager
•SharePoint Integration

       

To access the System Settings functionality, click on the Administration link in the left navigation. The Program Management page is displayed. Click on the System Settings link. The System Settings page is displayed.

 

All features available for management by administrative users are displayed. The features can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

To access the list of values for a particular feature, click on the Edit button next to the desired record in the System Settings section of the System Settings page. The landing page for the selected feature is displayed.
