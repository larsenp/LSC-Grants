Viewing Review Assignments by Committee

 

   

To view review assignments by a committee:

 

1.Select the Committee name next to the View Assignments By Committee field.
2.Click on the Go button. The Assignments by Committee page will open in a new window.

 

The Assignments by Committee page displays the current assignments for the committee members with the following columns:

 

•Reviewer
•Participant
•Task
•Easygrants ID
•Participant Organization
•Conflict/Preference
•Assignment

 

The results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

To close the Assignments by Committee page, click on the Close button.
