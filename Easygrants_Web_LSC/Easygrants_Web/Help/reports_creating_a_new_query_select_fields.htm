Selecting Query Fields

 

   

The Select Fields section of the Query Editor page allows the selection of the fields or columns that need to be displayed in the results. The list of available fields is comprised of Easygrants data fields that are typically selected in the query field group selected in step one.

 

To select the fields to be displayed in the query results:

 

1.Select if Duplicate Rows should be excluded. The Select Distinct checkbox limits the results to unique records only. For example, if the user selects First Name, Last Name and Primary Organization and Luke Dion were in the system twice with the same data for First Name, Last Name and Primary Organization, he would only show up once in the results set. If the user adds Committee Name and Luke is on two different committees, he will then show up twice since the data is not unique.

 

Excluding duplicate rows requires that the selected sort fields must be also in the selected fields list.

 

2.Select the fields to be displayed.

 

•To add a field, click on the desired field name from the Available list box and click on the Add button. The field(s) will be added to the Selected list box.
•To add several fields listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting fields in a row or the <CTRL> key if selecting fields throughout the list. Click on the Add button until all desired fields have all been added to the Selected list box.
•To remove previously selected fields from the Selected list box, select the field and click on the Remove button. The field(s) will be removed from the Selected list box.

 

3.Click on the Save button at the bottom of the page to save the selected fields.

 

Queries do not need to be saved in order to view the results.
