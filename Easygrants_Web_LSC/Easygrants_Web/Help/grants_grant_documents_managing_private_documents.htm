Managing Private Documents

   

The Private Documents section of the Grant Documents page allows users to upload documents to be shared only with other staff members. The private documents are not available to public users on the Grant Details page. Refer to Managing Public Documents for instructions on uploading documents to be shared with the grantee.

 

Private documents may include staff internal review comments or close-out/evaluation reports on the grant.

 

To add a private document :

 

1.Click the Add button in the Private Documents section of the Grant Documents page. The Private Documents Editor page is displayed.
2.Click the Browse button to select the file. The Choose File window is displayed to select the file from the local computer or network.
3.Click the Upload button to upload the file into Easygrants. The system will check that a file has been selected to upload and verify that the format is valid (e.g., .doc, .xls, .txt, .rtf., and .pdf). The file will be displayed in the Attachment Details section of the Private Documents Editor page.

 

•Click the View button next to the File/Document Name in the Attachment Details section to view the file in PDF in a new window.
•Click the Delete button next to the File/Document Name in the Attachment Details section to remove the uploaded file. Click the OK button in the confirmation pop-up to continue with the deletion. The file will no longer be displayed.
•Click the View button next to the Staff Access record in the Attachment Details section to view the file in the originally uploaded format in a new window.
•Click the Reset button to reset the file to be converted to PDF by the Easygrant's PDQueue technology. This will convert any uploaded files to PDF.

 

The Staff Access section of the Attachment Details section only displays if the user is logged in as a staff or administrative user.

 

4.Enter the file Description. To save the description, click on the Save button.
5.To return to the Grant Documents page, click on the Close button.

 

To view a private document in PDF, click the View button next to the desired record in the Private Documents section of the Grant Documents page. The file is displayed in PDF in a new window.

 

To delete a private document:

 

1.Click on the Delete button to the right of the desired record in the Private Documents section of the Grant Documents page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
