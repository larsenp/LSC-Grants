Grants

 

   

The grant record provides the functionality to view and manage related information in Easygrants including:

 

•Grant Details
•Activity
•Contacts
•Financials
•Documents
•Notes
•Demographics
•Media

 

Grant records are created either by a user applying for the funding opportunity from the link provided on the public user's Home page or by staff-initiated grants by using the Add Grant functionality on the Grants Easylist page.

 

To access the Grants section main page, click on the Grants link in the left navigation. The Grants Easylist page is displayed.
