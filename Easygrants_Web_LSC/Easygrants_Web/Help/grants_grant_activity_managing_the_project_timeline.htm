Managing the Project Timeline

 

   

The Project Timeline section displays the milestones/activities for the grant work that will be completed by the grantee. The initial project timeline can be entered as part of a task by the applicant and updated in assigned grantee reports (e.g., progress reports) by the grantee if the project timeline form is included in the task.

 

For example, the proposed work for a particular grant is to open a new public program with a publicly accessible office. One of the milestones/activities for the work is the opening of the office space.

 

Entering the project timeline separately from the uploaded proposal document (i.e., a MS Word document listing the full proposal) is used when it is key to include the proposed timeline as part of the grant agreement, separate tracking of the status of the activities is required, or staff need to frequently query on the information.

 

To add a milestone/activity for the project timeline:

 

1.Click Add in the Project Timeline section of Grant Activity page. The Project Timeline Editor page is displayed.
2.Enter the Milestone/Activity (required).
3.Enter the Description.
4.Enter the Expected Date (required).
5.Enter the Modified Date.
6.Select the Status (required).
7.Select if the milestone/activity is a deliverable.
8.Click on the Save button to save the information and remain on the Project Timeline Editor page. .
9.Click on the Save and Close button to save the information and return to the Grant Activity page.
10.To return to the Grant Activity page without saving the changes, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a milestone/activity for the project timeline, click on the Edit button next to the desired record in the Project Timeline section of the Grant Activity page. The Project Timeline Editor page is displayed with the selected record information. Use the steps above to complete the modification of the milestone/activity.

 

To delete a milestone/activity:

 

1.Click on the Delete button to the right of the desired record in the Project Timeline section of the Grant Activity page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record
