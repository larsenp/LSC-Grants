Assigning Tasks to Reviewers

 

   

The first step in managing the review stage is to set the review assignments. Assigning reviews tasks to a reviewer can be done in bulk using a set process or manually by the participant or reviewer.

 

The task that is assigned to the reviewer is the task selected on the Review Stage Editor page, e.g., Report Conflicts and Preferences, Submit Preliminary Scores and Comments, or Submit Panel Scores. The reviewer will receive the task for each participant submission (application, proposal, grantee report, renewal report) that they are assigned to review.

 

From the Review Stage Assignments page, staff can select the participants that a particular reviewer will be required to review.

 

To access the Review Stage Participants features:

 

1.Click on the Reviews link in the left navigation.
2.Click on the Review Processing link. The Review Cycles page is displayed.
3.Click on the Manage button for a specific review cycle. The Participant/Committee Pool page is displayed.
4.Click on the Review Stages tab in the review cycle. The Review Stages page is displayed.
5.Click on the Manage button for a specific review stage. The Review Stage Assignments page is displayed.
