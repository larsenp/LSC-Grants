Selecting the Review Cycle Participants

 

   

The first step in managing the review cycle is to select the review cycle participants. The participants include the tasks to be reviewed (e.g., Submit Application, Submit Final Report) and the review committees.

 

The available tasks are filtered by the selected Program and Funding Opportunities for the Review Cycle. The available committees are filtered to display all committees in Easygrants. This does not include subcommittees.
