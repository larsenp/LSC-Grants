Reminders

 

   

The reminders functionality allows staff and administrative users the ability to set up automatic correspondence to applicants and grantees in relation to assigned tasks.

 

Example: grantees are assigned an annual progress report with a specific due date. The granting organizations wants to send an e-mail to the grantee 30 days before the open date, 30 days before the due date, and 10 days after the due date. Three reminders would be set up for the Program, Funding Opportunity, and Funding Cycle. The staff or administrative user selects the Submit Progress Report task related to the hierarchy selected and the e-mail template to be used for the correspondence to the grantee. The user then enters the number of days, the before or after, and the due or open date for the task, in this case:

 

First Notification: 30 days + Before + Open Date

Second Notification: 30 days + Before + Due Date

Third Notification: 10 days + After + Due Date

 

The user enters the reminder name and description that displays on the Reminders page to help in locating specific reminders that need to be updated.

 

The Easygrants system runs a nightly process to find all matching records and sends the e-mail correspondence to the specified audience using an e-mail message template and merging in the appropriate data. The time of the nightly process is specified in the configuration of the system and will be set for all reminder correspondence sent by Easygrants.

 

To access the Reminders section, click on the Reminders link in the left navigation. The Reminders page is displayed.

 

The reminders are initially displayed in sets of 10 or less records. At the bottom of the section, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

The reminders can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.
