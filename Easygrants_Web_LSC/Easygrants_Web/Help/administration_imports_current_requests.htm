Current Import Requests

 

   

The Current Import Requests Listing page allows the user to view the request, remove the request from the import process, or commit to the request. The Current Import Requests Listing page also displays the status of a request. To see the latest status, click the Refresh button for your browser.

 

If the status is Ready to View, the file has been deemed a valid format and can be imported into Easygrants.

 

1.Click on the View Current Requests button in the Current Import Requests section. The Current Import Requests page is displayed.
2.Click on the View button to view the records to be imported. A pop-up window will display the file name, the date of import, and the records to be imported. If there are errors with the import file, the pop-up will display an error message. Click on the Close Window button to close the window.
3.Click on the Commit button to start the import process. A pop-up message is displayed. Select OK to continue with the import. To cancel the process, click Cancel. Clicking OK will redirect the user to the Historical Requests Listing page.

 

Once data has been imported into Easygrants it cannot be reversed.

 

To remove an Error status file:

 

1. Click on the Remove button next to the desired request in the Current Import Requests section. The page will refresh and no longer display the file.

 

If you are unsure as to why the file validation process failed, review the file formatting requirements for the file type you are attempting to import.
