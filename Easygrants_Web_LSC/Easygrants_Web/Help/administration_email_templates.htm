E-mail Templates

 

   

The e-mail templates provide the functionality to create standard e-mail communications that can be used as part of the workflow (automatically sent by the submission of a task or a particular outcome), for reminder communication, from the send e-mail action item related to a single record, or from the basic query tool for a group of records.

 

The templates are displayed in two separate sections, Add or Edit E-mail Templates and Edit System E-mail Templates. Any e-mail templates that are used in the workflow for a particular funding cycle can not be deleted and will display in the Edit System E-mail Templates. In order to delete those, the e-mail templates need to be removed from the workflow. The e-mail template will then display in the Add or Edit E-mail Templates section.

 

To access the E-mail Templates functionality, click on the E-mail Templates link in the left navigation. The E-mail Templates page is displayed.

 

The e-mail templates are initially displayed in sets of 10 or less records. At the bottom of the section, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

The e-mail templates can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Templates are configured with predefined information, including the From address, To address(es), CC address(es), BCC address(es), subject text and body text.  Like a mail merge document in a word processing program, the user can specify e-mail field merge codes. Easygrants will populate the merge codes with data upon sending the message.

 

For example: If the e-mail template indicates that all messages should go to [Person.PersonPrimaryEMail.ContactValue1]" the system will replace this code with applicant-specific data corresponding to this code.  In this example, each recipient's primary e-mail address will be added to the "To" field.

 

To view the merge codes that can be used in the e-mail template, click on the Query Fields and Merge Codes link in the instructional text on the E-mail Template Editor page. The Query Tool Field Groups and Merge Codes page is displayed in a new window. Select the Field Group from the drop-down and click on the Go button. The page will refresh and display the field names available with the related description and merge code.
