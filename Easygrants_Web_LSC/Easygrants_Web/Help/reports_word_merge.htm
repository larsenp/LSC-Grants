Word Merge

   

The Word Merge section of the Reports module allows the user to upload templates to be used when selecting "Word Merge" from the Query Tool. Documents added to the Word Merge section must be .doc or .dot files only.

 

To add a new template:

 

1.Click on the Add button in the Word Merge Documents section. The Word Merge Document Editor page is displayed.
2.Enter the Description (required).
3.Select the Field Group from the drop-down.
4.Select the Template Type from the drop-down.
5.Click on the Save button.
6.Click on the Browse button to select the file. The Choose File window is displayed to select the file from the local computer or network.
7.Click on the Upload button to upload the file into Easygrants. The system will check that a file has been selected to upload and verify that the format is valid (e.g., .doc or .dot). The file will be displayed in the Word Merge Document Details section. Click the View button to view the file in PDF in a new window.
8.Click on the Save button to save the file as a Word Merge Document.
9.To return to the Word Merge Documents page without saving the changes, click on the Close button.

 

To view a template, click on the View button next to the desired document. The document is displayed in a new window as the uploaded file type.

 

To edit a template, click on the Edit button next to the desired document. Use the steps above to complete the modification of the template.

 

To delete a template:

 

1.Click on the Delete button to the right of the desired document.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
