Home Page

 

   

The Home page contains links to the most commonly used features in the Easygrants Manager, such as personal task items and task items for your workgroup(s).

 

The Home page has three sections including:

 

•Options
•To Do - Application

 

The To Do - Application section will only be visible if the user has external Application tasks assigned.

 

• To Do - Review

 

The To Do - Review section will only be visible if the user has external Review tasks assigned.

 

•Personal Tasks
•Workgroup Tasks

 

To access the Home page, click on the Home link in the left navigation. The Home page is displayed.
