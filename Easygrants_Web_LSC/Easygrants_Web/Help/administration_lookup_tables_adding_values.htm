Managing Lookup Table Values

 

   

To access the list of values for a particular lookup table, click on the Edit button next to the desired record in the Lookup Tables section of the Lookup Table Management page. The Lookup Table List page is displayed.

 

If a Standard lookup table is selected for editing, all lookup table values are displayed.  If a Tiered lookup table is selected for editing, a Filter section is displayed to search for related lower-level categories and values.

 

To filter a Tiered lookup table, select a lower-level category from the drop-down list in the Filter section and click on the Find button. The page refreshes and displays all values related to the selected filter.

 

The lookup table values can be sorted by the column headers Description, Sort Order, and the Filter entity (Tiered type only). Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

To add new values to a lookup table:

 

1.Click on the Add button in Lookup Table Information section in the Lookup Table List page. The Lookup Table List Editor page is displayed.
2.Edit the Filter category (optional for Tiered type only).
3.Enter the Abbreviation (required).
4.Enter the Description (required).
5.Enter the Sort Order. If the table has been configured to use the sort order, the entered sort order will be the order the values are displayed in the checkbox or drop-down list. By default, tables are not configured to use the sort order and values are displayed in alpha or numeric order.
6.Select the Status. The Status will default to checked as Active. Active status values are available for selection by users of Easygrants. If the status is not checked as active, the value is no longer displayed for users to select.
7.Click on the Save button to save the information and remain on the Lookup Table List Editor page.
8.Click on the Save and Close button to save the information and return to the Lookup Table List page.
9.To return to the Lookup Table List page without saving the changes, click on the Close button.

 

To edit a lookup table value, click on the Edit button next to the desired record in the Lookup Table Information section of the Lookup Table List page. The Lookup Table List Editor page is displayed with the selected record information. Use the steps above to complete the modification of the lookup table value.

 

To remove a current lookup table value, edit the record and uncheck the Active status.
