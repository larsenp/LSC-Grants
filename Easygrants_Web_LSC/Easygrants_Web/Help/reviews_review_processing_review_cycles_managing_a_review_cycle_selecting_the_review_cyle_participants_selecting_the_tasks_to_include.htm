Selecting the Tasks to Include

 

   

To select the participants (tasks) to include in the review cycle:

 

1.In the Tasks to Include section, select the tasks that will be reviewed in the review cycle (required).

       

•The list of available tasks is comprised of all application and grantee reporting type tasks for the Program, Funding Opportunity, and Time Period/Award Cycle selected for the review cycle.
•To add a task, click on the task name from the Available list box and click on the Add button. The task(s) will be added to the Selected list box.
•To add several tasks listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting tasks in a row or the <CTRL> key if selecting tasks throughout the list. Click on the Add button until all desired tasks have all been added to the Selected list box.
•To remove previously selected tasks from the Selected list box, select the tasks and click on the Remove button. The task(s) will be removed from the Selected list box.

 

The tasks selected to be reviewed will be defaulted to be tasks with an outcome of submitted. To modify the tasks being reviewed by the outcome, e.g., to modify the selection to be only tasks that have been approved through internal review:

 

1.Click on the Edit Task Outcomes button in the Tasks to Include section of the Participant/Committee Pool page. The Participant Pool Editor page is displayed.
2.Click on the Edit button next to the desired task. The Participant Task Editor page is displayed.
3.Select the Outcomes for the Task (required).

 

Task Assignments are assigned an outcome that provides the details on the current decisions for the task. A public task assignment is updated with an outcome of Submitted when the assignee submits the task. If the task is reviewed internally prior to an external review cycle, the task assignment outcome may be updated to Approved by Staff. If only tasks that have been Approved by Staff should be a part of the review cycle, the Outcomes should be modified.

 

4.Click on the Save button to save the changes and remain on the Participant Task Editor page.
5.Click on the Save and Close button to save the changes and return to the Participant Pool Editor page.
6.To return to the Participant Pool Editor page without saving the changes, click on the Close button.
7.To return to the Participant/Committee Pool page, click on the Close button in the Participant Pool Editor page.

 

To add tasks to the review cycle from the Participant Pool Editor page:

 

1.Click on the Add button in the Participant Pool section of the Participant Pool Editor page. The Participant Task Editor page is displayed.
2.Select the Task from the drop-down (required). The page will refresh and display the associated outcomes.
3.Select the Outcomes for the Task (required).
4.Click on the Save button to save the changes and remain on the Participant Task Editor page.
5.Click on the Save and Close button to save the changes and return to the Participant Pool Editor page.
6.To return to the Participant Pool Editor page without saving the changes, click on the Close button.

 

Saving the page without entering required information or incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To delete tasks to the review cycle from the Participant Pool Editor page:

 

1.Click on the Delete button next to the desired task.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted task.

 

To view the selected participant pool:

 

1.Click on the View Selected Participant Pool link in the Tasks to Include section of the Participant/Committee Pool page. A new window will open displaying the results.

 

The results are displayed with the following columns:

               

•Participant
•Participant Organization
•Task
•Outcome
•Status

 

The results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.
