Upload Manager

 

   

The Upload Manager feature allows you to manage the upload types and templates used in Easygrants tasks. If a task is using the Uploads form, the user can select the upload types when uploading task related files.

 

For example, the Submit Proposal task is using the Uploads page as part of the submission requirements. For submission, the user must upload the Proposal and a Resume/CV. In addition, the user can also upload Other Supporting Documents. These three file types are added to the Upload Manager with the appropriate requirements for uploading.

 

In addition, the Proposal has a specific format that is used. A template with the format can be associated with the Proposal type upload allowing the user to download the template to use for the formatting.

 

Upload requirements are set by upload type including the file types allowed (.doc, .xls, etc.), page limits (only up to 2 pages), word limits, and character limits. The file size limit, 5 MB, is set at the system level so that all uploads have the same maximum file size.

 

 

To access the Upload Manager functionality, click on the Administration link in the left navigation. The Program Management page is displayed. Click on the Upload Manager link in the left navigation. The Upload Manager page is displayed.
