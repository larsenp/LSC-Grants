Person Contact Administration

 

   

The Contact Administration page allows the staff users to manage the login and password information for the Easygrants users. Administrative users will be able to add and manage all user information. Staff users will be able to add and manage public user information only.

 

To access the Contact Administration page, use the contacts search functionality to locate a person record. Select the action Contact Info and click on the Go button. The Person Contact Details page is displayed. Click on the Administration tab in the person record. The Contact Administration page is displayed.

 

If the person contact already has a user record, the system displays an Edit button to allow the administrative or staff user to manage the current information. If the person contact does not have a user record, an Add button is displayed.

 

To add or edit a user contact:

 

1.Click on the Add button in the Login Information section. The Login Editor page is displayed.
2.Select the Type if applicable. If the user creating the login is a staff user, the type will be Public and displayed as read only.
3.If the user should be required to change the password when logging in, check the checkbox.
4.If the user account should be locked, check the locked checkbox. If the user has tried to log in more than three times with the valid login but an incorrect password, the system will lock the account.
5.Click on the Create Login and Password button to create the user record. The Login will be created using the primary e-mail address for the person contact. If editing the record, this will be a Recreate button.
6.Click on the Send Password button to send an e-mail to the person contact with the login information.
7.Click on the Save button to save the information and remain on the Login Editor page.
8.Click on the Save and Close button to save the information and return to the Administration page.
9.To return to the Administration page, click on the Close button.

 

To send the password from the Administration page, click on the Send Password button. An e-mail will be sent to the contact with the login information.

 

To delete the user record:

 

1.Click the Delete button in the Login Information section.
2.Clicking the Delete button will display a pop-up message "Are you sure you want to delete this file?" with OK and Cancel options. Clicking Cancel will close the pop-up and keep the user on the page still displaying the file. Clicking OK will close the pop-up, refresh the page, and no longer display the selected file.
