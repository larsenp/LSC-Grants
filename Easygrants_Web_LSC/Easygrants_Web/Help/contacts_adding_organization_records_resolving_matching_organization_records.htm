Resolving Matching Organization Records

 

   

Upon saving the organization information, Easygrants checks for existing contact records based on the following:

 

•An exact match on the employer identification number (EIN)
•An exact match between the common organization name and the city and state of the organization

 

If an exact match is found, the user is redirected to the Matching Organizations page with the matching records displayed. To view the details for a matching record, click on the Common Name link. The Matching Organization Contact Details page is displayed in a new window providing a read only view of the  contact information for the organization record.

 

To continue adding the organization record, click on the Create a New Contact button at the bottom of the page. To return to the Add Organization Contact page without creating the contact, click on the Close button.

 

The Create a New Contact button will not be available if there is a duplicate match on the employer identification number (EIN). The employer identification number (EIN) must be unique in Easygrants.
