Searching for Application Tasks

 

   

To search for application type tasks:

       

1.Enter or select one or more search criteria contained in the Easylist Search section of the Applications Easylist page.

 

       The search filter criteria configured for the Easygrants system are:

 

•Easygrants ID
•Last Name
•First Name
•Organization
•E-mail
•Program
•Funding Opportunity
•Funding Cycle
•Task – (e.g., Submit Application)
•Role – (e.g., Applicant)
•Status – (e.g., Completed, Incomplete, Not Active)
•Outcome – The result or outcome of each action performed by a user (e.g., Once an applicant adds a reference to his/her application, possible outcomes are Reference Notified or Reference Not Notified for each application).

 

   

The Easylist functionality will assume an "And" for all the criteria. The more information you enter, the more the search results will be narrowed.

   

The Easylist functionality will assume a "contains" for text entry criteria and an "equals" for drop-down criteria.

 

2.Click on the Find button to execute the search. The page refreshes and displays a Search Results section containing the records matching the search.

 

       The columns that are displayed in the Search Results include:

 

•Action
•Easygrants ID
•Applicant
•Task
•Status
•Outcome
•Role

 

There will be a record returned for every task that is assigned to a person who comes up as a result of the search. For example, if John Smith is an applicant for two Programs/Funding Opportunities, then a search for Last Name = Smith will show two records for John Smith, each with a different Task (and possibly different outcomes).

 

The search results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Search results are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

3.Click on the Easylist Search section header to open the Search section again.
4.Click on the Clear button to clear any entered search criteria and start a new search.

 

To access the Action drop-down list options:

 

1.Select the desired activity in the Action drop-down list to the left of the appropriate record in the list.
2.Click on the Go button.

 

The options in the Action Menu for the Applications Easylist are detailed in the following table.

 

Action Item

Description

Contact Info

The Contact Info features allow you to manage contact details, task activities, uploaded documents, and reviewer information for the related applicant.

Login as this User

The Login as this User feature allows you to log in as the related applicant.

Send E-mail

The Send E-mail feature allow you to send custom or standardized e-mail messages to the selected applicant. You will be able to include additional recipients on the email request.

Tasks – Add New Task

The Tasks – Add New Task feature allows you to access the Assign Task Editor where you can view the current task and add a new task for the selected individual.

Tasks – Edit this Task

The Tasks – Edit this Task feature allows you to edit the details of a task assignment, such as the status, outcome, and task dates.

View Activity

The View Activity feature allows you to view that person's activity.

View PDF

The View PDF feature allows you to view the related PDF file for the selected task. If there is no related PDF file for the selected task, the system will display the message "Required PDF is not available."

View Project Activity

The View Project Tasks feature allows you to view detailed information on related assignments.

View Project Outcomes

The View Project Outcomes feature allows you to edit intermediate Funding Cycle outcomes and to select a final grantee Funding Cycle outcome.
