Grant Details

 

   

The Grant Details tab in the grant file allows users to manage the staff associated with the grant, key grant information, and grant assurance information.

 

The grant details are displayed in the following sections:

 

•Summary
•Grant Financial Summary
•Affiliated Staff
•Affiliated Staff History
•Grant Information
•Grant Award and Amendments
•Grant Status History
•Grant Assurances

 

To access the Grant Details page, use the Grants search functionality to locate a grant record. Select the Grant Info action and click on the Go button. The Grant Details page is displayed.

 

The Grant Details page can also be accessed by clicking on Easygrants ID links throughout Easygrants.
