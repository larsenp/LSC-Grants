Managing Links

   

The Managing Links section of the Grant Media page allows staff users to enter links (i.e. Internet Addresses) representative of media events related to the grant into the grant file.

 

To add a link :

 

1.Click the Add button in the Links section of the Grant Media page. The Links Editor page is displayed.
2.Enter the Description (required).
3.Enter the Link.
4.Select the Campaign Type, if applicable.
5.Click Save to save your data and remain on the Links Editor page.
6.Click Save and Close to save your data and return to the Grants Media page.
7.Click Close to return to the Grants Media page without saving your data.

 

 

To edit a link:

 

1.Click the Edit button next to the desired record in the Links section of the Grant Media page. The Grants Link Editor page is displayed. Use the steps above to edit the link.

 

To delete a link:

 

1.Click on the Delete button to the right of the desired record in the Links section of the Grant Media page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
