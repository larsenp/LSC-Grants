Setting Up the Default Grantee Reporting Schedule

 

   

The Default Grantee Reports Parameters for Funding Cycle section allows the user to create a standard grantee reporting schedule that will populate into the grant file when the grantee is awarded on the Review Stage Outcomes page in Review Processing. The schedule can be viewed and modified from the Financials tab of the grant file.

 

To create the default grantee report parameters for a Funding Cycle:

 

1.Click on the Add button in the Default Grantee Reports Parameters for Funding Cycle section of the Funding Cycle Details page. The Default Grantee Report Parameter Editor page is displayed with the Program, Funding Opportunity, and Funding Cycle as read only for reference purposes.
2.Select the Task from the drop-down.
3.Select the Award Term (the budget year for the report).
4.Enter the Task Dates, Start, Open, Due, Close and End dates.
5.Select the Status.
6.Select the Outcome.
7.Click on the Save button to save the information and remain on the Default Grantee Reports Parameter Editor page.
8.Click on the Save and Close button to save the information and return to the Funding Cycle Details page.
9.To return to the Funding Cycle Details page without saving the changes, click on the Close button.

 

The information entered here will pre-populate in the Reporting Schedule of the Financials Grant File tab upon awarding a grant record from the Review Stage Outcomes page.

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit the default grantee report parameters for a funding cycle, click on the Edit button next to the desired record in the Default Grantee Report Parameters for Funding Cycle section of the page. The Default Grantee Report Parameter Editor page is displayed with the selected record information. Use the steps above to complete the modification of the grantee report parameters.

 

To delete a default grantee report parameter for a funding cycle:

 

1.Click on the Delete button next to the desired record in the Default Grantee Report Parameters for Funding Cycle section.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
