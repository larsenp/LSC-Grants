Managing Workgroups

 

   

To add a workgroup:

 

1.Click on the Add button in the Workgroup Management section of the page. The Workgroups Editor page is displayed.
2.Enter the Workgroup Name (required).
3.Check if the workgroup is the default workgroup.

 

The default workgroup will be assigned any tasks where the task was assigned to a previously deleted workgroup or the primary staff person for the grant where a primary staff person has not been designated.

 

4.Select members for the Workgroup.

 

•The list of available members is comprised of all staff and administrative user types in Easygrants.
•To add a member, click on the name from the Available list box and click on the Add button. The member(s) will be added to the Selected list box.
•To add several members listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting members in a row or the <CTRL> key if selecting members throughout the list. Click on the Add button until all desired members have all been added to the Selected list box.
•To remove previously selected members from the Selected list box, select the members and click on the Remove button. The member(s) will be removed from the Selected list box.

 

5.Click on the Save button to save the information and remain on the Workgroups Editor page.
6.Click on the Save and Close button to save the information and return to the Workgroup Management page.
7.To return to the Workgroup Management page without saving the information, click on the Close button.

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a workgroup, click on the Edit button next to the desired record in the Workgroup Management section. The Workgroups Editor page is displayed with the selected record information. Use the steps above to complete the modification of the workgroup.

 

To delete a workgroup:

 

1.Click on the Delete button to the right of the desired record in the Workgroup Management section of the Workgroup Management page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
