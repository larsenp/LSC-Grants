Workgroups

 

   

The Workgroups feature allows the administrative user to assign staff or administrative user types to a group that does common tasks within Easygrants. Workgroups are associated with the task assignments in the Configuration section of Easygrants. Refer to Adding and Managing the Funding Cycle Workflow section for more information.

 

To access the Workgroups functionality, click on the Administration link in the left navigation. The Program Management page is displayed. Click on the Workgroups link in the left navigation. The Workgroup Management page is displayed.
