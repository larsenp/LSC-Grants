Finishing the Query

 

   

The Finish: Query Results section of the Query Editor page allows the user to view the query results and to save the query to be reused at a later time.

 

To view the results:

 

1.Select the desired view option.

 

•View Results (HTML)
•Excel
•E-mail
•Word Merge

 

Selecting an option will redirect the user to the appropriate page to complete the export or action (e-mail or Word Merge) as needed.

 

In order to verify that the results match the expected results, use the View Results (HTML) to get a quick glance of the data prior to exporting to Excel, using the results for a group e-mail, or using the results for a Word Merge.

 

To save a query:

 

1.Enter the Query Name (required).
2.Enter the Query Description.
3.Select the Query Type (required). Query Types are managed in the Lookup Table Manager by an administrative user.
4.Check if the Query is Private.

 

Marking a query as private will allow the viewing and editing of the query to only be done by the user that created the query.

 

5.Click on the Save button to save the query and remain on the Query Editor page.
6.To return to the Query/Report List page without saving the changes, click on the Close button.

 

If the query has been saved previously, a user can do a Save As function to copy the existing query selections and filters into a new query. Rather than modifying a query created by another user, use the Save As button and create a new query for modification.
