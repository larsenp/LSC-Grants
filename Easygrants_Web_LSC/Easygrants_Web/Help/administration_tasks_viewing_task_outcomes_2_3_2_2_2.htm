Modifying Task Dates

   

To modify the dates associated with the workflow tasks:

 

1.Click on the Modify Dates button in the Tasks section of the Workflow Builder Validation page. The Workflow Builder page is displayed with task name displayed as read only.
2.Modify the task dates, Start, Open, Due, Close, and End (required).

 

•Start Date: The date that the task will be displayed on the Home page. The task will not be enabled.
•Open Date: The date that the task is available to access.
•Due Date: The date that is published as the deadline for the task submission.
•Close Date: The date that the task is no longer available to access. It still displays on the Home page.
•End Date: The date the task no longer displays on the Home page.

 

3.Click on the Save button to save the information and remain on the Workflow Builder page.
4.Click on the Save and Close button to save the information and return to the Workflow Builder Validation page.
5.To return to the Workflow Builder Validation page without saving the changes, click on the Close button.

 

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.
