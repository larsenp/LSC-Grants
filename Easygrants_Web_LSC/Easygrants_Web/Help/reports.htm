Reports

 

   

The Reports functionality provides the staff and administrative users access to:

 

•Create and run basic queries using data collected in Easygrants
•Run custom reports (if applicable)
•Upload Word Merge documents to be used in the basic queries to create merged documents (e.g., Grant Award Letters)

 

 

To access the Reports functionality, click on the Reports link in the left navigation. The Query/Report List page is displayed.
