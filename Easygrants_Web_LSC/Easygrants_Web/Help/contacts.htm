Reassigning Planned Projects

 

   

To reassign a planned project to another staff or administrative user:

 

1.Click on the Reassign link for the desired planned project. The Reassign Project page is displayed.
2.Click on the Select button. The Person Selector is displayed.
3.Enter the last name of the staff or administrative user.
4.Click on the Find button.
5.Select the staff or administrative person.
6.Click on the Select button. To search for the person again, click on the Cancel button.
7.Click the Save button to save the changes and remain on the Planned Project Editor page.
8.Click on the Save and Close button to save the changes and return to the Home page.
9.To return to the Home page without saving the changes, click on the Close button.
