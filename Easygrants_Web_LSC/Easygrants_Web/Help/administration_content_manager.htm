Content Manager

 

   

The content manager provides the functionality to update task related instructional text providing clear directions to the assigned users (e.g. applicants, grantees, reviewers). Instructional content will be displayed based for the program, funding opportunity, and funding cycle selected. In addition, different instructional text can be displayed based on the status of the task assignment (e.g., Complete, Resubmit).

 

For example, for Program A's general funding opportunity, the Submit Proposal task for the 2008 cycle can have clear instructions for the project information page for an incomplete task requesting the project title and project description such as "In the fields below provide the title and description for your proposed project. Please limit the title to 80 characters and the description to 400 characters. Characters includes spaces. The description should provide a brief overview of what is included in your proposal write up." During the internal review, the staff may request a resubmission of the proposal asking for clarification in the description field. When the public user re-accesses the Submit Proposal task with the status of resubmit the text can be displayed as "It has been determined that the information you provided needs to be modified. Follow the instructions provided via e-mail by the funding organization to modify the project title and description."

 

The content entered in the Content Manager is displayed in the "Instructions" section on tasks in Easygrants. Instructional content is entered (and can be copied) per year.

 

       

To access the Content Manager functionality, click on the Administration link in the left navigation. The Program Management page is displayed. Click on the Content Manager link in the left navigation. The Content Manger page is displayed.
