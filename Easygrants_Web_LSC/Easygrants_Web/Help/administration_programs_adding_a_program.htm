Programs

   

All programs are listed in order of the Program ID in the Program Management section of the System Management page. Program setup and management is only accessible to Administrative users in Easygrants.

 

To access the Program Management section, click on the Administration link in the left navigation. The Configuration link in the left navigation will be highlighted and the user will be on the Program Management page which displays the Programs set up in Easygrants.

 

The results are displayed with the following columns:

               

•Program Name
•Program Display Name
•Program ID

 

The results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Results are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.
