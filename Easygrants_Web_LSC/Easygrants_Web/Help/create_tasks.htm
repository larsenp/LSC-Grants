Create Tasks

 

   

The Grant Tasks section allows users to create new tasks from within the Grant File.

 

To create a new task:

 

1.Select a task from the drop-down field.
2.Click Create.
3.Click OK in the pop-up window to create the task. Click Cancel to continue working within the grant without creating the new task.

 

Only tasks of Grant Tasks type and designated as Internal will be available for creation within the Grant Activity tab.

After clicking OK in the pop-up, the task will be created for the logged-in user. The user will see the newly created task on their Home Page and in the Task Assignment History section of the Grant Activity page.
