Managing Programs

 

   

To add a program:

 

1.Click on the Add button from the Program Management section of the Program Management Page. The Program Editor page is displayed.
2.Enter the Program Name (required).
3.Enter the Program Display Name (required).
4.Click on the Save button to save the information and remain on the Program Editor page.
5.Click on the Save and Close button to save the information and return to the Program Management page.
6.To return to the Program Management page without saving the changes, click on the Close button.

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To view the funding opportunity within a program, click the View button to the right of the desired program in the Program Management section of the Program Management page. The page refreshes and displays the Program Management page with the related Funding Opportunities displayed in the Funding Opportunity Management section. To return to the program level, click on the Program Management link in the History trail.

 

To edit a program, click on the Edit button next to the desired record in the Program Management section of the page. The Program Editor page is displayed with the selected record information. Use the steps above to complete the modification of the program.

 

To delete a program:

 

1.Click on the Delete button to the right of the desired program in the Program Management section of the page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

The Delete button is only available for Programs that do not have active task assignments.
