Running Basic Queries

 

   

The Basic Query section of the Reports module allows users to find and run saved queries and create new queries based on Easygrants data.

 

The Find Saved Queries section of the Query/Report List page allows the user to search for existing queries and perform specific actions on the search results including viewing the query results, exporting the results to Excel, and editing the query.

 

To Search for a Query:

 

1.Select the Query Type.
2.Click on the Go button.

 

- OR -

 

2.Type in the Query Name in the Search for Query Name Field.
3.Click on the Go button.

 

The matching results will display any private queries created by the logged in user and all public queries. Private queries created by other users will not be displayed.

 

The search results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Search results are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

To access the Action drop-down list options:

       

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu for saved queries are detailed in the following table.

 

Action Item

Description

Copy

The Copy option allows you to copy the selected queries to make another copy of it.

Delete

The Delete option removes the selected query from the Saved

Queries list and the system.

Edit

The Edit option opens the Query Tool page with all current field

and filter specifications for the saved query, allowing you to

change and save the report criteria as desired.

Export to E-mail Utility

The Export to E-mail Utility feature generates the selected query

and exports the query results to the Send E-mail page.

Export to Excel

The Export to Excel feature generates the selected query and

displays the query results in a new Web browser window in

Microsoft Excel format.

View Results

The View Results option generates the selected query and

displays the query results in a new Web browser window in HTML

format.

Word Merge

The Word Merge option allows you to merge data from the selected query with an uploaded Word Mail Merge Document.
