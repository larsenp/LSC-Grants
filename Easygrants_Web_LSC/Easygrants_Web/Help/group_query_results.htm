Group Query Results

 

   

The Group Query Results section of the Query Editor page allows you to group your query results by a particular field. This step is optional.

 

To group the query results by a particular field:

 

1.Select the field you would like to use for grouping purposes from the drop-down.
2.Click Add.

 

You can only group by one field. Selecting an additional field for grouping purposes will overwrite the previous field selected.

 

To remove the group:

 

1. Click the Delete button next to Selected Group by Field entry.
