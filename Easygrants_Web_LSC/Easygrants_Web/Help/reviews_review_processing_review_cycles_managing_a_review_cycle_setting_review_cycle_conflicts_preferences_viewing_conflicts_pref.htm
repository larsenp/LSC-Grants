Viewing Conflicts and Preferences

 

   

After conflicts and preferences are entered, the results are viewable in the View and Modify Conflicts and Preferences section.

 

To view the overall results in HTML:

 

1.Click on the View Conflicts and Preferences Results in HTML link in the View and Modify Conflicts and Preferences section on the Conflicts and Preferences Processing page. A new window will open displaying the results of the current conflicts and preferences.

 

The results are displayed with the following columns:

               

•Reviewer
•Reviewer Organization
•Committee
•Conflicts/Preferences
•Participant
•Participant Organization

 

The Conflicts/Preferences will be displayed as Able, Not Able, Preferred or Conflict.

 

2.Click the View Participant/Reviewer Research Category Matching Results in HTML link in the View Results or Set/Modify Requests and Conflicts Manually section on the Requests and Conflicts Processing page. A new window will open displaying the number of matches between the task selected research areas (set in the task submission) and the reviewer's research areas (set in the Person Contact Reviewer Info page).

 

The results are displayed with the following columns:

               

•Reviewer
•Reviewer Organization
•Committee
•Conflicts/Preferences
•Participant
•Participant Organization

 

The Research Category Matches will display the number of the matches. To view the actual categories, use the Query Tool.
