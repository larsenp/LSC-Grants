Person Attachments

 

   

Contact Attachments allows staff users to track related person attachments such as resumes, biography, photos, or publications.

 

 

To access the Contact Attachments page, use the contacts search functionality to locate a person contact record. Select the action Contact Info and click on the Go button. The Person Contact Details page is displayed. Click on the Attachments tab. The Contact Attachments page is displayed.

 

To add a person attachment :

 

1.Click the Add button in the File/Documents section of the Attachments page. The Attachments Editor page is displayed.
2.Click the Browse button to select the file. The Choose File window is displayed to select the file from the local computer or network.
3.Click the Upload button to upload the file into Easygrants. The system will check that a file has been selected to upload and verify that the format is valid (e.g., .doc, .xls, .txt, .rtf., and .pdf). The file will be displayed in the Attachments Details section of the Attachments Editor page.

 

•Click the View button next to the File/Document Name in the Attachment Details section to view the file in PDF in a new window.
•Click the Delete button next to the File/Document Name in the Attachment Details section to remove the uploaded file. Click the OK button in the confirmation pop-up to continue with the deletion. The file will no longer be displayed.
•Click the View button next to the Staff Access record in the Attachments Details section to view the file in the originally uploaded format in a new window.
•Click the Reset button to reset the file to be converted to PDF by the Easygrant's PDQueue technology. This will convert any uploaded files to PDF.

 

The staff access section of the Attachment Details section only displays if the user is logged in as a staff or administrative user.

 

4.Enter the file Description. To save the description, click the Save button.
5.To return to the Attachments page, click the Close button.

 

To view a person attachment in PDF:

 

1.Click the View button next to the desired record in the File/Documents section of the Contact Attachments page. The file is displayed in PDF in a new window.

 

To delete a person attachment from the main Attachments page:

 

1.Click on the Delete button to the right of the desired record in the File/Documents section of the Attachments page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.
