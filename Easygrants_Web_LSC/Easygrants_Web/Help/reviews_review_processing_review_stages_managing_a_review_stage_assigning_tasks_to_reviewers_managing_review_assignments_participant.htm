Managing Review Assignments by Participant

 

   

Review assignments can be viewed, set, and updated by participant within the View and Modify Assignments section of the Review Stage Assignments page of the review stage.

 

To view an assignment by the participant:

 

1.Select the Participant name next to the View and Modify Assignments by Participant field.
2.Click on the Go button. The Assignments by Participant page is displayed.

 

The Assignments by Participant page displays the current assignments (including the number of assignments) in the Review Assignments by Participant section with the following columns:

 

•Participant
•Reviewer
•Task
•Easygrants ID
•Participant Organization
•Conflicts/Preference
•Assignment Type
•Options

 

The results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

In the Review Assignments by Participant section, new assignments can be added or existing assignments can be modified or deleted.

 

To return to the Review Stage Assignments page for the review stage, click on the Close button.

 

To add a review assignment for a participant:

 

1.Click on the Add button on the Review Assignments by Participant section. The Assignments by Participant Editor page is displayed.
2.Select the Reviewer from the drop-down (required).

 

If a participant has already been assigned to the reviewer, the reviewer will not be available in the drop-down.

 

3.Select the Review Assignment Type (required). The type will default to None. Select Primary, Secondary, or Reader.
4.Click on the Save button to save the information and remain on the Assignments by Participant Editor page.
5.Click on the Save and Close button to save the information and return to the Assignments by Participant page.
6.To return to the Assignments by Participant page without saving changes to the information, click on the Close button.

 

Modifying existing review assignments can be done in bulk or one at a time.

 

To edit all review assignments for a participant:

 

This option is only available after at least one assignment has been made. If there are no assignments, the Edit All Assignments link will not be displayed.

 

1.Click on the Edit All Assignments link above the Add button in the Review Assignments by Participant section. The Assignments Editor by Participant page is displayed.
2.Select the Reviewer Assignment Type from the drop-down (required). The type will default to Primary. Select Primary, Secondary, or Reader.

 

The reviewer assignment type selected will be the type for all assignments made using Edit All Assignments process.

 

3.Select the reviewers to be assigned (required).

 

•The list of available reviewers is comprised of all members of the committees selected for the review cycle.
•To add a reviewer, click on the reviewer name from the Available list box and click on the Add button. The reviewer(s) will be added to the Selected list box.
•To add several reviewers listed in succession in the Available list box to the list, select the first item and then use the <SHIFT> key for selecting reviewers in a row or the <CTRL> key if selecting reviewers throughout the list. Click on the Add button until all desired reviewers have all been added to the Selected list box.
•To remove previously selected reviewers from the Selected list box, select the reviewers and click on the Remove button. The reviewer(s) will be removed from the Selected list box.

 

4.Click on the Save button to save the information and remain on the Assignments Editor by Participant page.
5.Click on the Save and Close button to save the information and return to the Review Assignments by Participant page.
6.To return to the Assignments by Participant page without saving changes to the information, click on the Close button.

 

To edit a single review assignment for a participant:

 

1.Click on the Edit button next to the desired review assignment. The Assignments by Participant Editor page is displayed. The reviewer name will be displayed as read only.
2.Select the Review Assignment Type (required). The type will default to Primary. Select Primary, Secondary, or Reader.
3.Click on the Save button to save the information and remain on the Assignments by Participant Editor page.
4.Click on the Save and Close button to save the information and return to the Assignments by Participant page.
5.To return to the Assignments by Participant page without saving changes to the information, click on the Close button.

 

Deleting existing review assignments can be done in bulk or one at a time.

 

To delete all review assignments for a participant:

 

1.Click on the Delete All Assignments link in the Review Assignments by Participant section.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the review assignments for the participant.

 

To delete a single review assignment for a participant:

       

1.Click on the Delete button next to the desired review assignment.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted assignment record.
