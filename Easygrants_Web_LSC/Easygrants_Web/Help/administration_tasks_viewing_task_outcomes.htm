Application Workflow Builder

 

   

The Application Workflow Builder allows the user to select application type tasks (e.g., Submit Letter of Intent, Submit Proposal), third party type tasks (e.g., Submit Letter of Reference), and internal review tasks (e.g., Review Letter of Intent) to be completed as part of the workflow.

 

Application type tasks are created in the Task Builder. Refer to the Task section for more details.

 

To access the Application Workflow Builder page:

 

1.Click on the View button next to the desired Funding Cycle in the Funding Cycle Management section of the page.
2.The page refreshes and displays the workflow builder tabs for each grant lifecycle phase starting with the Eligibility Workflow Builder.
3.Click on the Application tab.

 

 

The Application Workflow Builder is comprised of the following steps:

 

1.Selecting the tasks to be completed in the application phase either by the public user or by internal staff.
2.Determining the application workflow based on the outcome of the related tasks.
3.Determining the third party (references) workflow based on the outcome of the related tasks.
