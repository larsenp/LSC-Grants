Viewing the Variance Report

 

   

From the Review Stage Scores and Comments page, the variance report can be generated and viewed. The variance report will display the divergences between reviewers scores for a participant within a particular committee.

 

To view the variance report for a review stage:

 

1.Select the Committee next to the Run Variance Report.
2.Click on the Go button. The Variance Report page is displayed in a new window.
3.Click on the Close button to close the new window.
