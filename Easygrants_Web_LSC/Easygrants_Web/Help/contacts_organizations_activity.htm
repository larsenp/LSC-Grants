Organization Contact Activity

 

   

The Activity tab in the Organization record displays grant records that the organization is associated with either as the grantee or in another role, e.g., Fiscal Agent.

 

The Grant Projects section of the Organization Contact Activity displays all grants for which the organization is the grantee. The grant records will be displayed in descending order by the Easygrants ID with the following columns:

 

•Action
•Easygrants ID
•Project Title
•Funding Cycle
•Program
•Status
•Outcome

 

The search results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Search results are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

To access the Action drop-down list options:

 

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu are detailed in the following table.

 

Action Item

Description

Grant Info

The Grant Info feature allows you to manage the details of a grant, including people affiliations, budget information, progress report information, and payment details.

View Project Activity

The View Project Tasks feature allows you to view detailed information on related assignments.

 

To add other grant project affiliations:

 

1.Click on the Add button in the Other Grant Project Affiliation section of the Contact Activity page. The Other Grant Project Affiliation Editor page is displayed.
2.Select the Grant Project. Clicking on the Select button will display the Grantee Project Selector.
3.Select the Role.
4.Click on the Save button to save the information and remain on the Other Grant Project Affiliation Editor page.
5.To return to the Activity page without saving the information, click on the Close button.
