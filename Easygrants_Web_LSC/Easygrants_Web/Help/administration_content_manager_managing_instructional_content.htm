Managing Instructional Content

 

   

To add instructional content for a particular task:

 

1.Select the Program Name. The page refreshes and displays the Funding Opportunity field.
2.Select the Funding Opportunity. The page refreshes and displays the Funding Cycle field.
3.Select the Funding Cycle. The page refreshes and displays the Task field.
4.Select the Task. The page refreshes displaying the Add button and any previously entered content for the task in the Pages section.
5.Click on the Add button to add new instructional content. The Content Editor page is displayed.
6.Select the Page Name (required).
7.Select the Task Status (required).

 

Page content must be added separately for each status, incomplete, complete, and resubmit. Not Active status tasks do NOT appear in the To Do lists and do not require instructional content.

8.Enter the Content (required) in plain text or with HTML tags.
9.Click on the Preview button to view the instructional content as it will appear on the task page.
10.Click on the Save button to save the data and remain on the Content Editor page.
11.To return to the Content Manager page without saving the changes, click on the Close button.

 

Saving the page without entering the required information or with incorrect formatting will display page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit instructional content for a page, click on the Edit button next to the desired record in the Pages section of the Content Manager page. The Content Editor page is displayed with the selected record information. Use the steps above to complete the modification of the instructions.

 

To remove instructional content for a page, click on the Edit button and remove the text in the Content field. Click on the Save button to save the changes. The instructions section for the task page and status will display as blank.

 

To preview the instructional content for a page, click on the Preview button next to the desired record in the Pages section of the Content Manager page. The content will be displayed in a new window.
