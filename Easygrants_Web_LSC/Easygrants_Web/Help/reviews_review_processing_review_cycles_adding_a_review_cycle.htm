Creating and Modifying Review Cycles

 

   

To add a review cycle:

 

1.Click on the Add button in the Review Cycles section of the Review Cycles page. The Review Cycle Editor page is displayed.
2.Select the Program (required).
3.Select the Funding Opportunity(s) (required).
4.Enter the Review Cycle Name (required).
5.Select the Time Period/Award Cycle (required).

 

The tasks available to be reviewed will be filtered by the selected time period/award cycle. The time period/award cycle is based on the selection made at the funding cycle level.

 

6.Enter the review cycle Description.
7.Click on the Save button to save the review cycle details and remain on Review Cycle Editor page.
8.Click on the Save and Continue button to save the review cycle details and continue with the addition of details for review cycle. Refer to the section Managing a Review Cycle for additional information.
9.To return to the Review Cycles page without saving changes to the information, click on the Close button.

 

Saving the page without entering required information or with incorrect formatting will display a page level validation in red text at the top of the page. Required fields are marked with a red asterisk.

 

To edit a review cycle, click on the Edit button next to the desired record in the Review Cycles section of the Review Cycles page. The Review Cycle Editor page is displayed with the selected record information. Use the steps above to complete the modification of the review cycle.

 

To delete a review cycle:

 

1.Click on the Delete button to the right of the desired record in the Review Cycles section of the Review Cycles page.
2.Click on the OK button in the confirmation pop-up to continue with the deletion. The page refreshes and no longer displays the deleted record.

 

If there are active review tasks, the review cycle cannot be deleted.
