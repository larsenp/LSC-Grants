Searching for a Reviewer

 

   

The Reviewers Easylist page allows you to quickly search for and view information on person contacts designated as reviewers. To access the Reviewers Easylist page, click on the Reviews link in the left navigation and then on the Reviewers link.

 

To search for a reviewer:

 

1.Enter or select one or more search criteria contained in the Easylist Search section of the Reviewers Easylist page.

 

       The search filter criteria configured for the Easygrants system are:

               

•Last Name
•First Name
•Organization
•E-mail
•Committee
•Reviewer Status

 

   

The Easylist functionality will assume an "And" for all the criteria. The more information you enter, the more the search results will be narrowed.

   

The Easylist functionality will assume a "contains" for text entry criteria and an "equals" for drop-down criteria.

 

2.Click on the Find button to execute the search. The page refreshes and displays a Reviewers Search Results section containing the list of reviewers matching the search.

       

       The columns that are displayed in the Search Results include:

 

•Action
•Reviewer
•Reviewer Organization
•Committee

 

There will be a record returned for every committee that is assigned to a reviewer who comes up as a result of the search. For example, if John Smith is a reviewer on two committees, then a search for Last Name = Smith will show two records for John Smith, each with a different committee.

 

The search results can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Search results are displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10. To access all the records, if applicable, click on the Show All link.

 

Using the Action Menu

 

The Action drop-down list in the Search Results section of the Reviewers Easylist page contains options to manage a reviewer’s information.

 

To access the Action drop-down list options:

 

1.Select the desired activity in the Action drop-down list to the left of the desired record in the list.
2.Click on the Go button.

 

The options in the Action Menu for the Reviewers Easylist are detailed in the following table.

 

Action Item

Description

Contact Info

The Contact Info features allow you to manage contact details, task activities, group memberships, uploaded documents, and

nominator information for the selected individual.

Login as this User

The Login as this User feature allows you to log in as the selected individual.

Send E-mail

The Send E-mail pages allow you to send custom or standardized e-mail messages to the selected individual. You are able to add additional recipients to the email.

View Person Tasks

The View Person's feature allows you to view the selected individual's tasks.
