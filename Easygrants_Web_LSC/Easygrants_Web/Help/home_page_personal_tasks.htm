Personal Tasks

 

   

All tasks assigned to you are listed in the Personal Tasks section of the Home page. Personal Tasks are assigned to the primary staff person affiliated with a grant file or to a workgroup.

 

The columns that are displayed in the Personal Tasks include:

 

•Task Creation Date
•Task
•Easygrants ID
•Organization
•Grant Status
•Primary Person
•Reassign

 

The personal task listing can be sorted by the column headers. Clicking once will sort the list in ascending alpha or numeric order. Clicking on the column header a second time will sort the list in descending alpha or numeric order.

 

Personal Tasks are initially displayed in sets of 10 or less records. At the bottom of the page, the total number of records will be listed (e.g., Viewing Records 1 - 10 of 30). To access the next set of records, if applicable, click on the Next 10 link. To access the last set of records, if applicable, click on the Last 10 link. To view all records, if applicable, click on the Show All link.

 

Only incomplete tasks will be displayed in your Personal Tasks section.
