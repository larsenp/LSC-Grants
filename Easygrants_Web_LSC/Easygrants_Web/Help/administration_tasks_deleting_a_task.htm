Grantee Reporting Workflow Builder

 

   

The Grantee Reporting Workflow Builder allows the user to select grantee report type tasks (e.g., Submit Progress Report, Submit Final Report), third party type tasks (e.g., Submit Performance Review), and internal review tasks (e.g., Review Progress Report) to be completed as part of the workflow.

 

Grantee Report type tasks are created in the Task Builder. Refer to the Task Builder section for more details.

 

To access the Grantee Reporting Workflow Builder page:

 

1.Click on the View button next to the desired Funding Cycle in the Funding Cycle Management section of the page.
2.The page refreshes and displays the workflow builder tabs for each grant lifecycle phase starting with the Eligibility Workflow Builder.
3.Click on the Grantee Reporting tab.

 

 

The Grantee Reporting Workflow Builder is comprised of the following steps:

 

1.Selecting the tasks to be completed in the grantee reporting phase either by the public user or by internal staff.
2.Determining the grantee reporting workflow based on the outcome of the related tasks.
3.Determining the third party (references) workflow based on the outcome of the related tasks.
